## 需要手动下载的数据

### 4. 12.5m DEM - ALOS PALSAR

**下载地址**: https://www.eorc.jaxa.jp/ALOS/en/dataset/palsar_l2.htm

**下载步骤**:
1. 访问JAXA EORC网站
2. 注册账号并申请数据访问权限
3. 搜索以下瓦片:
   - 瓦片ID: `N029E100`
   - 覆盖区域: 29°N-30°N, 100°E-101°E
4. 下载 `HV` 或 `HH` 极化数据
5. 需要进行 Radiometric Terrain Correction (RTC) 处理

**替代方案**:
- **Copernicus DEM GLO-30**: 已下载，30m分辨率
- **SRTM 1 Arc-Second**: 30m，从 https://earthexplorer.usgs.gov/ 下载
- **ALOS World 3D (AW3D30)**: 30m，从 https://www.eorc.jaxa.jp/ALOS/en/dataset/aw3d30/ 下载

### 5. 土地覆盖 - ESA WorldCover 2021

**下载地址**: https://esa-worldcover.org/en/data-access

**下载步骤**:
1. 访问ESA WorldCover网站
2. 点击 "Data Access"
3. 使用地图选择器选择覆盖以下区域的瓦片:
   - 经度: 100.846°E ~ 101.124°E
   - 纬度: 29.876°N ~ 30.154°N
4. 下载 10m 分辨率的土地覆盖分类数据

**数据说明**:
- 10m分辨率全球土地覆盖
- 包含11个类别:
  - 10: Tree cover
  - 20: Shrubland
  - 30: Grassland
  - 40: Cropland
  - 50: Built-up
  - 60: Bare/sparse vegetation
  - 70: Snow and ice
  - 80: Permanent water bodies
  - 90: Wetlands
  - 95: Mangroves
  - 100: Moss and lichen

### 6. 水系数据补充 - HydroSHEDS

**下载地址**: https://www.hydrosheds.org/

**推荐数据**:
- **HydroRIVERS**: 河流矢量数据
- **HydroSHEDS DEM**: 水文校正DEM
- **HydroBASINS**: 流域边界

**下载步骤**:
1. 访问 HydroSHEDS 网站
2. 选择 "Download" 页面
3. 下载覆盖亚洲区域的数据集
4. 使用GIS软件裁剪到分析区域

### 7. 土壤湿度 - ERA5-Land

**下载地址**: https://cds.climate.copernicus.eu/

**下载步骤**:
1. 注册 Copernicus Climate Data Store (CDS) 账号
2. 安装 Python API: `pip install cdsapi`
3. 配置API密钥（保存到 `~/.cdsapirc`）
4. 运行以下代码:

```python
import cdsapi

c = cdsapi.Client()

c.retrieve(
    'reanalysis-era5-land-monthly-means',
    {
        'product_type': 'monthly_averaged_reanalysis',
        'variable': [
            'soil_moisture_level_1',
            'soil_moisture_level_2',
            'soil_moisture_level_3',
            'soil_moisture_level_4',
        ],
        'year': '2023',
        'month': ['01', '02', '03', '04', '05', '06',
                  '07', '08', '09', '10', '11', '12'],
        'time': '00:00',
        'area': [30.154, 100.846, 29.876, 101.124],  # North, West, South, East
        'format': 'netcdf',
    },
    'era5_land_soil_moisture.nc')
```

**数据说明**:
- 0.1° (~9km) 分辨率
- 月平均数据
- 包含4层土壤湿度:
  - Level 1: 0-7cm
  - Level 2: 7-28cm
  - Level 3: 28-100cm
  - Level 4: 100-289cm

### 8. OSM小路数据补充

已下载的 `osm_roads.osm` 包含所有道路类型。如需筛选小路:

**在QGIS中筛选**:
1. 打开 `osm_roads.osm`
2. 选择 "lines" 图层
3. 使用表达式筛选:
   ```
   "highway" IN ('path', 'footway', 'track', 'steps', 'pedestrian')
   ```

**在ArcGIS中**:
1. 导入OSM数据
2. 按 `highway` 字段筛选

## 数据整合建议

### 坐标系统一
所有数据应转换为统一坐标系:
- 推荐: **EPSG:4326** (WGS84) 或 **EPSG:32647** (UTM 47N)

### 分辨率统一
- DEM: 重采样到相同分辨率（如30m或12.5m）
- 土地覆盖: 重采样到DEM分辨率
- 土壤湿度: 插值到更高分辨率

### 数据格式
- 栅格数据: GeoTIFF (.tif)
- 矢量数据: Shapefile (.shp) 或 GeoJSON (.geojson)
