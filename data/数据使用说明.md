## 📋 数据文件清单（按用途分类）

### 一、地形数据（DEM）

| 文件名 | 说明 | 格式 | 分辨率 | 用途 |
|--------|------|------|--------|------|
| `dem_12m_resampled.tif` | **12.5米DEM（推荐使用）** | GeoTIFF | 12.5m | 坡度、坡向、曲率分析 |
| `dem_30m_clipped.tif` | 30米DEM（精确裁剪） | GeoTIFF | 30m | 大范围地形分析 |
| `copernicus_dem_30m.tif` | 原始Copernicus DEM | GeoTIFF | 30m | 备用数据 |
| `yajiang_gesigou_srtmgl1_30m_1000px_bbox.tif` | 原始SRTM DEM | GeoTIFF | 30m | 参考数据 |
| `yajiang_srtmgl1_30m_bbox.tif` | SRTM DEM（大范围） | GeoTIFF | 30m | 参考数据 |

**Python读取方法：**
```python
import rasterio

# 读取DEM
with rasterio.open('dem_12m_resampled.tif') as src:
    dem = src.read(1)  # 读取第一个波段
    transform = src.transform
    crs = src.crs
    print(f"尺寸: {src.width} x {src.height}")
    print(f"分辨率: {src.res}")
```

### 二、道路数据

| 文件名 | 说明 | 格式 | 用途 |
|--------|------|------|------|
| `osm_roads.shp` | **道路矢量数据（推荐）** | Shapefile | 路网分析、通行能力评估 |
| `osm_roads.dbf` | 属性表 | Shapefile组件 | 道路属性（类型、名称） |
| `osm_roads.shx` | 空间索引 | Shapefile组件 | 加速查询 |
| `osm_roads.prj` | 坐标系定义 | Shapefile组件 | 坐标系统信息 |
| `osm_roads.cpg` | 编码定义 | Shapefile组件 | 字符编码 |
| `osm_roads.osm` | 原始OSM数据 | XML | 数据备份 |

**道路类型说明：**
- `trunk` - 主干道（61条）
- `primary` - 一级公路（28条）
- `secondary` - 二级公路（6条）
- `residential` - 居民区道路（56条）
- `track` - 小路/土路（37条）⭐ 通行性分析重点
- `footway` - 人行道（16条）
- `path` - 步道（6条）⭐ 通行性分析重点

**Python读取方法：**
```python
import geopandas as gpd

# 读取道路数据
roads = gpd.read_file('osm_roads.shp')
print(f"道路数量: {len(roads)}")
print(f"道路类型: {roads['highway'].unique()}")

# 筛选小路
tracks = roads[roads['highway'].isin(['track', 'path', 'footway'])]
```

### 四、土壤湿度数据 ⭐ 新下载

| 文件名 | 说明 | 格式 | 用途 |
|--------|------|------|------|
| `era5_land_soil_moisture_2023.nc` | **ERA5-Land土壤湿度** | NetCDF | 土壤湿润程度分析 |

**数据说明：**
- **来源**: ECMWF ERA5-Land 再分析数据
- **时间**: 2023年全年月平均
- **空间分辨率**: 0.1°（约9公里）
- **包含变量**:

## 🔧 Python 实验脚本示例

### 完整的数据加载示例

```python
"""
雅江格西沟通行能力分析 - 数据加载示例
"""

import rasterio
import geopandas as gpd
import xarray as xr
import numpy as np
from pathlib import Path

# 数据目录
DATA_DIR = Path("data")

def load_dem():
    """加载DEM数据"""
    print("="*60)
    print("加载DEM数据")
    print("="*60)

    dem_path = DATA_DIR / "dem_12m_resampled.tif"

    with rasterio.open(dem_path) as src:
        dem = src.read(1)
        transform = src.transform
        crs = src.crs
        bounds = src.bounds

        print(f"尺寸: {src.width} x {src.height}")
        print(f"分辨率: {src.res}")
        print(f"坐标系: {crs}")
        print(f"高程范围: {dem.min():.2f}m ~ {dem.max():.2f}m")

    return dem, transform, crs, bounds

def load_roads():
    """加载道路数据"""
    print("\n" + "="*60)
    print("加载道路数据")
    print("="*60)

    roads_path = DATA_DIR / "osm_roads.shp"

    roads = gpd.read_file(roads_path)

    print(f"道路数量: {len(roads)}")
    print(f"道路类型: {roads['highway'].unique()}")

    # 筛选小路（通行性分析重点）
    tracks = roads[roads['highway'].isin(['track', 'path', 'footway'])]
    print(f"小路/步道数量: {len(tracks)}")

    return roads, tracks

def load_waterways():
    """加载水系数据"""
    print("\n" + "="*60)
    print("加载水系数据")
    print("="*60)

    water_path = DATA_DIR / "osm_waterways.shp"

    waterways = gpd.read_file(water_path)

    print(f"水系要素数量: {len(waterways)}")

    return waterways

def load_soil_moisture():
    """加载土壤湿度数据"""
    print("\n" + "="*60)
    print("加载土壤湿度数据")
    print("="*60)

    sm_path = DATA_DIR / "era5_land_soil_moisture_2023.nc"

    ds = xr.open_dataset(sm_path)

    print(f"变量: {list(ds.data_vars)}")
    print(f"时间范围: {ds.time.values[0]} ~ {ds.time.values[-1]}")
    print(f"空间范围:")
    print(f"  经度: {ds.longitude.values.min():.3f} ~ {ds.longitude.values.max():.3f}")
    print(f"  纬度: {ds.latitude.values.min():.3f} ~ {ds.latitude.values.max():.3f}")

    # 获取表层土壤湿度
    soil_moisture = ds['volumetric_soil_water_layer_1']

    return ds, soil_moisture

def main():
    """主函数"""
    print("="*60)
    print("雅江格西沟通行能力分析 - 数据加载")
    print("="*60)

    # 加载所有数据
    dem, transform, crs, bounds = load_dem()
    roads, tracks = load_roads()
    waterways = load_waterways()
    ds, soil_moisture = load_soil_moisture()

    print("\n" + "="*60)
    print("数据加载完成!")
    print("="*60)
    print(f"\n可用变量:")
    print(f"  dem - DEM高程数据 (numpy数组)")
    print(f"  roads - 所有道路 (GeoDataFrame)")
    print(f"  tracks - 小路/步道 (GeoDataFrame)")
    print(f"  waterways - 水系 (GeoDataFrame)")
    print(f"  soil_moisture - 土壤湿度 (xarray DataArray)")

if __name__ == "__main__":
    main()
```
