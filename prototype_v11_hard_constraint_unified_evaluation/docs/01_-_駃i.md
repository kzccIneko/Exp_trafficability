# v10 公式护照：逐项来源、含义与使用边界

## 1. DEM 方向坡度分解

DEM 表示地形表面：

```math
z=z(x,y)
```

局部梯度为：

```math
\nabla z_i=(p_i,q_i)
```

车辆行驶方向单位向量为：

```math
\mathbf{u}_d=(\cos\theta_d,\sin\theta_d)
```

车辆横向方向单位向量为：

```math
\mathbf{n}_d=(-\sin\theta_d,\cos\theta_d)
```

纵坡角：

```math
\alpha_{\parallel}(i,d)=\arctan(\nabla z_i\cdot\mathbf{u}_d)
```

横坡角：

```math
\alpha_{\perp}(i,d)=\arctan(|\nabla z_i\cdot\mathbf{n}_d|)
```

人话解释：纵坡表示车沿前进方向是在上坡还是下坡；横坡表示车身左右方向倾斜多少。

理论依据：GIS 成本距离中的 vertical factor 明确强调坡度对移动代价的影响取决于移动者如何遇到坡度；不能在方向起作用时简单把坡度作为静态成本面。

## 2. 上坡牵引利用率

```math
\rho_{up}(i,d)=\frac{\tan(\max(\alpha_{\parallel}(i,d),0))}{\tan(\alpha_{grade})+\varepsilon}
```

分子是当前上坡坡度需求；分母是最大可接受爬坡能力场景。

该式不是完整发动机/传动系统牵引力模型。因为当前没有轮端扭矩、胎压、传动比、速度等实测数据，所以用可接受爬坡角作为场景参数。

## 3. 下坡制动利用率

```math
\rho_{down}(i,d)=\frac{\tan(\max(-\alpha_{\parallel}(i,d),0))}{\mu_b(i)+\varepsilon}
```

分子表示下坡导致车辆需要制动和附着控制的需求；分母是空间化制动附着能力。

v10 的改进是把常数 `mu_b` 改成空间栅格 `mu_b(x,y)`。湿润地表、水系附近、软弱土壤区域会降低 `mu_b`，相同下坡条件下更容易接近制动限制。

## 4. 侧翻稳定性利用率

车辆静态稳定因子：

```math
SSF=\frac{B}{2h_c}
```

侧翻稳定性利用率：

```math
\rho_{roll}(i,d)=\frac{\tan|\alpha_{\perp}(i,d)|}{B/(2h_c)+\varepsilon}
```

分子是横坡对车辆稳定性的需求；分母是轮距和质心高度决定的静态稳定能力。

注意：这是简化静态表达，不预测动态侧翻事故；质心高度未公开，因此用 `h_c/H` 做敏感性分析。

## 5. 侧滑附着利用率

```math
\rho_{slide}(i,d)=\frac{\tan|\alpha_{\perp}(i,d)|}{\mu_s(i)+\varepsilon}
```

分子是横坡导致车辆向侧下方滑动的需求；分母是空间化横向附着能力。

v10 的改进是把常数 `mu_s` 改成空间栅格 `mu_s(x,y)`。

## 6. 地表附着参数空间化

v10 使用 PTF-Lite，即基于 GIS 的地表附着参数转换模型。

土地覆盖给出基础附着能力：

```math
\mu_{base}(x,y)=Lookup(LandCover(x,y))
```

TWI 表示地形湿润倾向：

```math
TWI=\ln\left(\frac{A_s}{\tan\beta+\varepsilon}\right)
```

综合湿润倾向：

```math
W(x,y)=w_1TWI_n(x,y)+w_2W_{water}(x,y)+w_3S_{soil}(x,y)
```

最终空间化附着能力：

```math
\mu_s(x,y)=clip[\mu_{s,base}(x,y)(1-\gamma_sW(x,y)),\mu_{s,min},\mu_{s,max}]
```

```math
\mu_b(x,y)=clip[\mu_{b,base}(x,y)(1-\gamma_bW(x,y)),\mu_{b,min},\mu_{b,max}]
```

人话解释：地表类别决定基础附着能力；地形湿润、水系附近和软弱土壤倾向会降低附着能力。

该模型不是完整土壤传递函数，不声称反演真实摩擦系数。默认附着表和 `gamma` 是工程情景参数，需要做消融和敏感性分析。

## 7. 主导限制和路径代价

```math
\rho_{max}(i,d)=\max(\rho_{up},\rho_{down},\rho_{roll},\rho_{slide})
```

```math
c(i,d)=1+\rho_{max}(i,d)
```

```math
J(\Gamma)=\sum_{e\in\Gamma}l_e[1+\rho_{max,e}]
```

解释：每一小段路都看车辆最接近哪一种限制。路径越长，代价越大；路径越多路段接近车辆能力限制，代价也越大。

## 8. 当前不进入主模型的内容

通过角几何限制不进入 30m DEM 主模型。Jeep 轴距约 3m，而 30m DEM 不能解析石块、小沟坎、局部坡顶和底盘托底条件。通过角只作为高分辨率扩展说明。
