# 车辆越野通行能力文献分析提取报告

## 文献来源

本报告基于以下两篇文献的系统分析：

1. **文献1**: "Using crowdsourced fitness tracker data to model the relationship between slope and travel rates" (Campbell et al., 2019) - 坡度与移动速度关系模型
2. **文献2**: "Speed Limit Safety of Expressway Curves Based on the Critical State Evaluation Model of Vehicle Side Rollover" (Xie, 2018) - 车辆坡度+转向对速度的影响分析

---

## 一、坡度-速度关系模型

### 1.1 经典坡度-移动速度函数

#### (1) Tobler 徒步函数 (THF, 1993)

**公式**:
```
v = 1.6 × exp(-3.5 × |tan(θ) + 0.05|)
```

其中：
- v: 速度 (m/s)
- θ: 坡度角 (度)

**特点**:
- 假设最佳移动速度出现在轻微下坡 (-2.86°)
- 最大速度约 1.67 m/s
- 采用双指数形式，峰值尖锐

#### (2) Naismith 规则 (1892)

**公式**:
```
v = 0.726 / tan(θ)    (上坡)
```

**Langmuir 修正 (1984)**:
- 陡下坡 (θ ≤ -12°): `v = 0.722 / tan|θ|`
- 缓下坡 (-12° < θ ≤ -5°): `v = 0.722 / tan|θ|`

**特点**:
- 平坡速度约 1.40 m/s
- 仅考虑上坡，下坡需修正

#### (3) Davey 等人模型 (1994)

**公式**:
```
v = v₀ × exp(-3.5 × |tan(θ)|)
```

其中：
- v₀: 平地可持续行走速度 (基准值约 1.40 m/s)

**特点**:
- 提供了个体差异调整因子

#### (4) Rees 多项式模型 (2004)

**公式**:
```
v = 0.75 - 0.09 × tan(θ) - 14.6 × (tan(θ))²
```

**特点**:
- 钟形曲线形式
- 峰值不明显

#### (5) Campbell 等人模型 (2017)

**公式**:
```
v = 1.662 - 5.191×10⁻² × θ - 1.127×10⁻³ × θ²
```

**特点**:
- 最大速度 1.67 m/s 出现在 -2.3°
- 基于37名受试者的越野实验

#### (6) Irmischer & Clarke 模型 (2018)

**公式**:
```
v = 0.11 + 6.0 × exp(-0.5 × ((θ + 2.86) / 8.5)²)
```

**特点**:
- 高斯曲线形式
- 基于200名军校学员GPS追踪

### 1.2 本研究提出的概率分布模型

本研究基于29,928名用户的421,247次徒步/跑步GPS轨迹数据，提出了三种概率分布函数模型：

#### (1) 拉普拉斯模型 (Laplace)

**公式**:
```
v_Laplace = c × (1/(2b)) × exp(-|θ-a|/b) + d + e×θ
```

#### (2) 高斯模型 (Gauss)

**公式**:
```
v_Gauss = c × (1/(b×√(2π))) × exp(-0.5×((θ-a)/b)²) + d + e×θ
```

#### (3) 洛伦兹模型 (Lorentz) - 推荐使用

**公式**:
```
v_Lorentz = c × (1/π) × (b/((θ-a)² + b²)) + d + e×θ
```

**参数说明**:
- a: 曲线中心位置 (最佳速度坡度)
- b: 曲线宽度参数
- c: 速度范围缩放因子
- d: 基础速度偏移
- e: 不对称性修正系数

**模型性能对比**:

| 模型 | R² (平均) | MAE (平均) |
|------|-----------|------------|
| Lorentz | 0.958 | 0.078 m/s |
| Laplace | 0.953 | 0.088 m/s |
| Gauss | 0.949 | 0.090 m/s |

**关键发现**:
- 最佳速度坡度范围: -1.41° 到 -4.00°
- Lorentz模型在5th-95th百分位表现最佳
- Laplace模型在10th-25th百分位表现较好
- 上坡速度普遍低于下坡速度（不对称性）

## 二、车辆动力学模型

### 2.1 车辆侧翻临界模型

#### 临界侧翻速度公式

**水平曲线段**:
```
v_r = sqrt((2h_i + B) × g × R × cos(α) / (2h - B×i))
```

其中：
- v_r: 临界侧翻速度
- h_i: 车辆重心高度
- B: 轮距
- g: 重力加速度 (9.81 m/s²)
- R: 曲线半径
- α: 横向坡度角
- i: 超高值

**坡道曲线段**:
```
v_r = sqrt(2h_i + B) × g × R × cos(θ) / (2h - B×i)
```

其中：
- θ: 纵向坡度角

### 2.2 车辆侧滑临界模型

#### 临界侧滑速度公式

**水平曲线段**:
```
v_s = sqrt((g × cos(α) + g × sin(α)) × R / (cos(α) - sin(α)))
```

简化形式:
```
v_si = sqrt((g + g×i) × R / (1 - i))
```

其中：
- v_s: 临界侧滑速度
- μ: 轮胎与路面横向摩擦系数

**坡道曲线段**:
```
v_si = sqrt((g + g×i) × R × cos(θ) / (1 - i))
```

### 2.3 力学分析

#### 车辆受力方程

**侧向力平衡**:
```
m×a_y = F_y1 + F_ya + m×g×cos(α)×sin(α)
```

**法向力平衡**:
```
m×g×cos(α)×cos(α) + m×a_y×sin(α) = F_z1 + F_za
```

**力矩平衡 (绕纵轴旋转)**:
```
cos(θ) × (m×a_y×cos(α)×h - 0.5×m×a_y×B×sin(α) - m×g×h×cos(α)×sin(α) + F_z1×B - 0.5×m×g×B×cos(α)) = 0
```

## 三、转向性能分析

### 3.1 曲线半径对临界速度的影响

**关键结论**:
- 曲线半径越大，临界侧翻/侧滑速度越高
- 临界侧翻速度随横向坡度值增大而提高（在有限范围内）
- 临界侧滑速度随摩擦系数增大而提高

### 3.2 纵向坡度的影响

**关键结论**:
- 纵向坡度对侧翻和侧滑临界速度影响较小
- 下坡路段更容易发生侧滑事故

### 3.3 速度一致性评价

**评价标准**:
- |ΔV₈₅| ≤ 10 km/h: 速度一致性很好
- 10 < |ΔV₈₅| < 20 km/h: 速度一致性一般
- |ΔV₈₅| ≥ 20 km/h: 速度一致性差，需调整设计

## 四、能耗模型相关

### 4.1 坡度对能耗的影响

根据文献1的研究：
- 上坡需要更高的能量输出
- 下坡需要制动以维持安全
- 随着坡度增加，移动速度普遍下降

### 4.2 速度-能耗关系

**关键发现**:
- 低百分位（5th）: R² = 0.941 (Lorentz)
- 高百分位（95th）: R² = 0.914 (Lorentz)
- 步行速度比跑步速度更一致

## 六、应用建议

### 6.1 坡度-速度模型选择

**推荐使用Lorentz模型**:
- 在5th百分位表现最佳（最接近平均步行速度）
- 在30th-95th百分位范围表现优异
- R²平均值最高 (0.958)
- MAE平均值最低 (0.078 m/s)

### 6.2 车辆安全速度限制

**速度限制制定原则**:
1. 曲线半径是影响车辆稳定性的最重要因素
2. 横向坡度值在有限范围内增大可提高侧翻临界速度
3. 摩擦系数是影响侧滑临界速度的关键因素
4. 纵向坡度对临界速度影响较小

### 6.3 实际应用

**速度限制方案示例**:
- 小型客车: 80-120 km/h (根据曲线半径)
- 大型卡车: 80-100 km/h (根据曲线半径)
- 相邻路段速度差应控制在20 km/h以内
