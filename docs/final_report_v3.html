引言

越野通行能力（off-road trafficability）是指车辆在无铺装路面的自然地形条件下安全、有效行驶的能力。随着自动驾驶技术、应急救援需求和军事机动性研究的快速发展，越野通行能力建模已成为地理信息科学、车辆工程和军事科学等多学科交叉的前沿课题。如何基于地理信息数据——特别是数字高程模型（DEM）、遥感影像和土壤数据库——建立科学、定量的通行能力评估模型，是当前研究的核心问题。

本报告围绕"基于地理信息数据的越野通行能力建模分析方法"这一课题，从研究背景与标准化框架、单要素量化建模、多因子综合评估、路径规划应用、现有不足与前沿趋势五个层面展开系统性综述，并在此基础上提出创新方法设计。报告共分八章：第1章介绍研究背景与数据基础；第2章阐述各通行阻碍因子的提取与量化方法；第3章讨论多因子综合评估模型；第4章分析通行能力图在路径规划中的应用；第5章审视现有研究不足与前沿趋势；第6章提出方向敏感的创新建模方法；第7章设计实验验证方案；第8章总结创新点并给出论文建议。

1. 越野通行能力建模的研究背景、数据基础与标准化框架

1.1 定义与起源：从军事需求到学科范式

越野通行能力的概念起源于20世纪中期的军事机动性研究。第二次世界大战期间，盟军在北非和欧洲战场的大规模装甲作战暴露了车辆在非结构化地形中的机动性限制问题。战后，美国陆军工程兵水道实验站（WES）率先开展了系统的土壤-车辆力学研究，奠定了越野通行能力建模的理论基础。

核心定义：越野通行能力是指特定车辆在给定地形和环境条件下，能够安全通过且不发生不可接受的性能降级（如速度过低、油耗过高、结构损伤）的能力。这一概念包含三个关键维度：

  • 几何通过性：车辆物理尺寸能否通过地形的几何约束（如坡度、沟宽、路宽）
  • 力学通过性：车辆动力系统能否克服地形施加的阻力（如坡度阻力、土壤变形阻力）
  • 稳定性通过性：车辆在行驶过程中能否保持稳定（如防侧翻、防打滑）

从学科发展脉络看，越野通行能力建模经历了三个阶段：经验模型阶段（1940s–1970s），以WES的车辆指数（Vehicle Cone Index, VCI）为代表，主要依赖实地试验和经验公式；物理模型阶段（1970s–2000s），以Bekker-Wong土壤力学理论和NRMM为代表，引入了系统的力学分析；数据驱动阶段（2000s至今），以GIS空间分析、遥感反演和机器学习方法为代表，实现了从局部试验到大范围评估的跨越。

1.2 理论基础与学科交叉特性

越野通行能力建模是一个典型的多学科交叉问题，其理论基础涉及以下核心领域：

（1）土壤力学与车辆地面力学

Bekker (1960) 提出的压力-沉陷理论是越野通行能力建模的经典基础。该理论将土壤对车辆的响应简化为法向压力与沉陷深度的关系：

$$p = \left(\frac{k_c}{b} + k_\phi\right) z^n$$

其中 $p$ 为法向压力（kPa），$b$ 为加载面短边长度（m），$k_c$ 为土壤内聚模数，$k_\phi$ 为土壤摩擦模数，$z$ 为沉陷深度（m），$n$ 为沉陷指数。Wong (2001) 在此基础上进一步发展了车辆地面力学理论，建立了包含牵引力、行驶阻力和沉陷效应的完整分析框架。

（2）地形分析与数字高程模型

DEM是越野通行能力建模最重要的空间数据基础。基于DEM可以提取坡度、坡向、曲率、地形湿度指数（TWI）等关键地形参数。Zevenbergen & Thorne (1987) 提出的偏四次曲面拟合方法是地形参数提取的经典方法，通过3×3窗口的9个高程点拟合局部曲面，进而解析计算各阶地形参数（详见第2章）。

（3）多准则决策分析（MCDA）

越野通行能力评估本质上是一个多因素综合决策问题。Saaty (1980) 提出的层次分析法（AHP）是应用最广泛的权重确定方法，通过构建判断矩阵和一致性检验实现主观判断的定量化。Zadeh (1965) 的模糊集理论为处理通行能力评估中的不确定性提供了数学工具。

（4）空间分析与地理信息系统

GIS技术为越野通行能力评估提供了空间数据管理、分析和可视化平台。栅格数据模型天然适合表达连续的通行能力表面，叠加分析（overlay analysis）为多因子综合提供了计算框架。遥感技术的发展使得大范围、多时相的地表覆盖和土壤湿度数据获取成为可能。

1.3 主要数据源及其特点

越野通行能力建模所需的数据可分为地形数据、地表覆盖数据、土壤数据、水系数据和气象数据五大类：

数据融合的挑战：不同数据源在空间分辨率、时间分辨率、坐标系统和分类标准上存在显著差异。例如，土壤数据（1km）与地形数据（30m）之间存在尺度差异，直接叠加分析会导致伪精度问题。尺度转换（scaling）和空间降尺度（downscaling）是数据融合的关键技术。

1.4 国际标准化框架：NRMM与NG-NRMM

北约参考机动性模型（NATO Reference Mobility Model, NRMM）是国际上最具影响力的越野通行能力评估标准化框架。该模型由美国陆军工程兵水道实验站于1970年代开发，经过多次迭代升级，已成为北约成员国军事车辆机动性评估的通用标准。

NRMM的核心架构：

  • 第一性原理模块：基于Bekker-Wong土壤力学理论，计算车辆在特定土壤条件下的牵引力和行驶阻力
  • 经验回归模块：基于大量实地试验数据，建立车辆性能参数（速度、油耗）与地形因子的统计关系
  • 限制条件模块：定义几何通过性限制（最大坡度、最大侧坡、最小路宽等）

NG-NRMM的革新：2024年发布的新一代北约参考机动性模型（Next-Generation NRMM, NG-NRMM）实现了从经验模型向物理-数据融合模型的根本性转变。NG-NRMM的主要改进包括：

  • 引入高保真车辆动力学仿真，替代简化经验公式
  • 集成LiDAR点云数据的精细地形分析
  • 支持动态土壤含水率条件
  • 预测误差从NRMM的75.1%大幅降低至6.2%

然而，NG-NRMM对详细车辆参数的高度需求限制了其通用性。在缺乏精确车辆模型的场景下（如民用越野、应急救援），仍需依赖简化的评估方法。

1.5 应用领域扩展：从军事到民用

越野通行能力建模的应用已从最初的军事机动性评估扩展到多个民用领域：

（1）自动驾驶越野

自动驾驶车辆在非结构化环境中的导航需要精确的通行能力评估。代价图（cost surface）作为路径规划的核心输入，直接影响规划路径的安全性和效率。

（2）应急救援

自然灾害（地震、洪水、泥石流）发生后，道路网络可能严重损毁。越野通行能力评估可用于规划救援车辆的最优通行路线，提高救援效率。近年来，无人机辅助的快速地形评估技术为应急场景提供了新的数据获取手段。

（3）农林作业

大型农业机械和林业采运车辆在田间和林区作业时面临复杂的越野条件。通行能力评估可辅助作业路线规划，减少土壤压实和设备损伤。精准农业中的变量管理也需要考虑土壤承载力的空间变异。

（4）资源勘探与基础设施建设

在偏远地区进行矿产勘探、油气管线建设和风电场选址时，越野通行能力评估是施工可行性和成本估算的重要依据。

1.6 当前研究的主要挑战与争议

尽管越野通行能力建模研究已取得显著进展，但仍面临以下核心挑战：

（1）权重确定的主观性

多因子综合评估中，各因子权重的确定高度依赖专家判断。不同专家的判断可能差异显著，导致评估结果的可重复性不足。AHP、熵权法、CRITIC法等方法各有优劣，尚无公认的最优方案。

（2）尺度效应与数据分辨率

DEM分辨率对地形参数提取和通行能力评估结果有显著影响。30m分辨率DEM可能遗漏关键地形细节（如小型冲沟、岩石露头），而1m分辨率DEM的数据量和计算成本又难以承受。如何在精度和效率之间取得最优平衡是持续争议的焦点。

（3）动态性与时效性

土壤含水率、积雪覆盖、植被生长等环境因素随时间变化，静态的通行能力图无法反映这些动态变化。建立考虑时变因素的动态通行能力模型是重要的研究方向，但数据获取和模型复杂度仍是主要障碍。

（4）车辆-环境耦合

不同车辆类型（轮式/履带、两驱/四驱、轻型/重型）在相同地形条件下的通行能力差异显著。现有模型多针对特定车辆类型，缺乏通用的车辆-环境耦合框架。

（5）验证困难

越野通行能力评估结果的实地验证成本高、风险大、可重复性差。基于OSM路网的间接验证方法（详见第6章）为这一问题提供了新的解决思路，但其有效性和适用范围仍需进一步验证。

2. 通行阻碍因子的提取与单要素量化建模

越野通行能力评估的核心在于将复杂的自然环境要素转化为可量化的参数，并建立各要素与车辆通行性之间的数学关系。本章系统阐述坡度、地表粗糙度、地表覆盖、土壤承载力、水系障碍及气象因素等六大通行阻碍因子的提取方法与量化模型，为后续多因子综合评估奠定基础。

2.1 坡度因子的提取与量化

坡度是影响越野通行能力最直接的地形因子，直接影响车辆的牵引力需求、行驶稳定性及能耗。基于数字高程模型（DEM）的坡度提取主要有差分法和拟合曲面法两类。差分法通过计算相邻栅格的高程差与水平距离之比获得坡度值，计算公式为：$S = \arctan(\Delta h / d)$，其中$\Delta h$为高程差，$d$为水平距离。拟合曲面法则通过局部窗口内的高程点拟合二次曲面，再计算该曲面的最大梯度方向及坡度值，对噪声具有更好的抑制效果。

2.1.1 Zevenbergen & Thorne (1987) 曲面拟合法

Zevenbergen & Thorne (1987) 提出的偏四次多项式曲面拟合法是地形参数提取的经典方法。该方法采用3×3窗口（9个网格点）的高程数据，通过偏四次多项式拟合地表曲面：

$$z = Ax^2y^2 + Bx^2y + Cxy^2 + Dx^2 + Ey^2 + Fxy + Gx + Hy + I$$

其中9个系数与3×3高程矩阵的关系（设网格间距为 $\Delta$）如下：

$$I = z_5$$

$$G = \frac{z_4 - z_6}{2\Delta}$$

$$H = \frac{z_2 - z_8}{2\Delta}$$

$$D = \frac{z_4 + z_6 - 2z_5}{2\Delta^2}$$

$$E = \frac{z_2 + z_8 - 2z_5}{2\Delta^2}$$

$$F = \frac{z_1 - z_3 - z_7 + z_9}{4\Delta^2}$$

$$A = \frac{(z_1 + z_3 + z_7 + z_9) - 2(z_2 + z_4 + z_6 + z_8) + 4z_5}{4\Delta^4}$$

$$B = \frac{(z_1 + z_3 - z_7 - z_9) - 2(z_2 - z_8)}{4\Delta^3}$$

$$C = \frac{(-z_1 + z_3 - z_7 + z_9) - 2(z_4 - z_6)}{4\Delta^3}$$

基于上述系数，可直接通过解析公式计算坡度、坡向和曲率等关键地形参数：

坡度（Slope）：

$$\text{SLOPE} = \arctan\sqrt{G^2 + H^2}$$

坡向（Aspect）：

$$\theta = \arctan\left(\frac{-H}{-G}\right)$$

剖面曲率（Profile Curvature）——沿最大坡度方向的曲率：

$$\text{PROFC} = -2(D\cos^2\theta + E\sin^2\theta + F\cos\theta\sin\theta) = \frac{-2(DG^2 + EH^2 + FGH)}{G^2 + H^2}$$

平面曲率（Planform Curvature）——垂直于最大坡度方向的曲率：

$$\text{PLANC} = 2(D\sin^2\theta + E\cos^2\theta - F\sin\theta\cos\theta) = \frac{2(DH^2 + EG^2 - FGH)}{G^2 + H^2}$$

该方法的优势在于：使用偏四次多项式而非二次多项式，可更好表示复杂地形；所有二阶导数系数（D, E, F）可通过3×3窗口直接计算；坡度、坡向、曲率均可用解析公式表达，无需迭代。

2.1.2 坡度-速度关系模型

坡度对通行速度的影响可通过坡度-速度比例关系模型量化。根据《军事地形学》中的经验数据，车辆在不同坡度下的机动速度呈现非线性下降趋势。研究表明，坡度每增加1°，轮式车辆的通行速度约降低3%-5%，而履带式车辆的降速幅度相对较小(Construction of dynamic trafficability map)。具体而言，当坡度超过15°时，普通轮式车辆的通行能力急剧下降；超过25°时，大多数车辆难以通行。履带式车辆的坡度阈值通常可提高至30°-35°。

Campbell et al. (2019) 基于29,928名用户的421,247次徒步/跑步GPS轨迹数据，系统比较了6种经典坡度-移动速度函数模型，并提出了3种概率分布函数模型。

（1）Tobler 徒步函数（THF, 1993）：

$$v = 1.6 \times \exp(-3.5 \times |\tan(\theta) + 0.05|)$$

其中 $v$ 为速度（m/s），$\theta$ 为坡度角（度）。该模型假设最佳移动速度出现在轻微下坡（-2.86°），最大速度约1.67 m/s，采用双指数形式，峰值尖锐。

（2）Naismith 规则（1892）及其Langmuir修正（1984）：

$$v = \frac{0.726}{\tan(\theta)} \quad \text{（上坡）}$$

Langmuir修正：陡下坡（$\theta \leq -12°$）时 $v = 0.722 / \tan|\theta|$。该模型仅考虑上坡情形，平坡速度约1.40 m/s。

（3）Davey 等人模型（1994）：

$$v = v_0 \times \exp(-3.5 \times |\tan(\theta)|)$$

其中 $v_0$ 为平地可持续行走速度（基准值约1.40 m/s），提供了个体差异调整因子。

（4）Rees 多项式模型（2004）：

$$v = 0.75 - 0.09 \times \tan(\theta) - 14.6 \times (\tan(\theta))^2$$

该模型呈钟形曲线形式，峰值不明显。

（5）Campbell 等人模型（2017）：

$$v = 1.662 - 5.191 \times 10^{-2} \times \theta - 1.127 \times 10^{-3} \times \theta^2$$

最大速度1.67 m/s出现在-2.3°，基于37名受试者的越野实验。

（6）Irmischer & Clarke 模型（2018）：

$$v = 0.11 + 6.0 \times \exp\left(-0.5 \times \left(\frac{\theta + 2.86}{8.5}\right)^2\right)$$

该模型采用高斯曲线形式，基于200名军校学员GPS追踪数据。

2.1.3 概率分布坡度-速度模型

Campbell et al. (2019) 在上述经典模型基础上，进一步提出了三种概率分布函数模型，其中Lorentz模型表现最优（推荐使用）：

Laplace模型：

$$v_{\text{Laplace}} = c \times \frac{1}{2b} \times \exp\left(-\frac{|\theta - a|}{b}\right) + d + e \times \theta$$

Gauss模型：

$$v_{\text{Gauss}} = c \times \frac{1}{b\sqrt{2\pi}} \times \exp\left(-\frac{(\theta - a)^2}{2b^2}\right) + d + e \times \theta$$

Lorentz模型（推荐）：

$$v_{\text{Lorentz}} = c \times \frac{1}{\pi} \times \frac{b}{(\theta - a)^2 + b^2} + d + e \times \theta$$

其中参数含义为：$a$ 为曲线中心位置（最佳速度坡度），$b$ 为曲线宽度参数，$c$ 为速度范围缩放因子，$d$ 为基础速度偏移，$e$ 为不对称性修正系数。

2.2 地表粗糙度因子的提取与量化

地表粗糙度反映地表的起伏破碎程度，是影响车辆行驶平稳性和振动载荷的关键因子。

2.2.1 地形位置指数（TPI）

地形位置指数（TPI）是量化地表粗糙度的常用指标，其计算公式为：

$$\text{TPI} = Z_0 - \bar{Z}$$

其中 $Z_0$ 为目标点高程，$\bar{Z}$ 为邻域平均高程。基于TPI值可将地形划分为山脊、上坡、中坡、平坦坡地、下坡和谷地六类，不同地形类型对轮式和履带式车辆的通行能力影响系数各异(Construction of dynamic trafficability map)。

2.2.2 局部标准差与曲率

基于DEM的粗糙度指标还包括局部标准差、曲率等。局部标准差通过计算窗口内高程值的标准差来量化地表起伏强度，公式为：

$$\sigma = \sqrt{\frac{\sum(Z_i - \bar{Z})^2}{n}}$$

其中 $Z_i$ 为窗口内各点高程，$n$ 为点数。曲率则通过计算高程曲面的二阶导数获得，反映地表的凹凸形态。

地形粗糙度指数（TRI）的计算公式为（Riley et al., 1999）：

$$\text{TRI} = \sqrt{\frac{\sum(Z_{ij} - Z_{00})^2}{n}}$$

其中 $Z_{ij}$ 为邻域单元格高程，$Z_{00}$ 为中心单元格高程，$n$ 为邻域单元格数。

2.2.3 基于分形布朗运动（fBm）的粗糙度模型

刘华军等（2006）提出基于分形布朗运动（fBm）模型的地形粗糙度量化方法。fBm模型通过Hurst指数 $H$ 描述地表的自相似性和粗糙程度：

$$\log E\left[|f(x+\Delta x, y+\Delta y) - f(x,y)|\right] = H \log d + \log C$$

其中 $d = \sqrt{\Delta x^2 + \Delta y^2}$，$H$ 为Hurst指数，$D_f = n + 1 - H$ 为分形维数。

Hurst指数的计算公式为：

$$H = \frac{\log E[|\Delta f|] - \log C}{\log d}$$

其中 $H$ 越大表示表面越平滑，$H$ 越小表示表面越粗糙。

2.2.4 横滚角（Roll）计算

横滚角是评估车辆侧倾稳定性的关键参数。刘华军等（2006）提出使用局部地形方差估计横滚角：

$$\text{Roll} = \text{var}(f_{ij}) = \frac{1}{N-1} \sum_{(x_m, y_n) \in N_{ij}} \left[f(x_m, y_n) - \bar{f}_{ij}\right]^2$$

2.3 地表覆盖因子的提取与量化

地表覆盖类型直接影响车辆的附着力、滚动阻力及通行障碍。常用的土地覆盖分类数据包括GlobeLand30（30米分辨率）、MODIS（250米-1公里分辨率）等全球产品，以及基于Sentinel-2等卫星影像的区域精细分类数据。

2.3.1 遥感植被指数公式

土壤调节植被指数（SAVI）：

其中 $L = 0.465$ 为土壤调节系数。

土壤含水量指数（SMI）：

$$\text{SMI} = \frac{\text{SWIR1} - \text{TIR1}}{\text{SWIR1} + \text{TIR1}}$$

2.4 土壤承载力因子的提取与量化

土壤承载力是决定车辆是否陷入地面的关键因子，通常通过锥指数（CI）和重塑锥指数（RCI）进行量化。锥指数表示土壤抵抗锥形探头贯入的能力，单位为kPa。重塑锥指数则考虑了车辆反复碾压对土壤的重塑效应，更能反映实际通行条件下的土壤强度。

2.4.1 重塑锥指数（RCI）计算公式

Knight (1961) 提出的RCI通用计算公式为：

$$\text{RCI} = \exp\left[a' - b' \ln(\text{MC})\right]$$

其中 MC 为土壤质量含水率，$a'$ 和 $b'$ 为与土壤类型相关的经验系数。Mason & Baylot (2016) 提出了等价形式：

$$\text{RCI} = \exp\left[a - b \ln\left(\frac{V_w}{V_s} \cdot \frac{r_w}{r_s}\right)\right]$$

其中 $V_w$、$r_w$ 为水的体积和容重，$V_s$、$r_s$ 为土壤的体积和容重(Construction of dynamic trafficability map)。

2.4.2 车辆锥指数（VCI）计算公式

车辆锥指数（VCI）是车辆特定的最低土壤强度要求。陶泽兴等（2025）给出了轮式和履带车辆的VCI计算公式：

轮式车辆VCI：

$$\text{VCI}_{\text{LUN}} = 11.48 + 0.2 M_{I\text{LUN}} - \frac{39.2}{M_{I\text{LUN}} + 3.74}$$

轮式车辆机动指数：

$$M_{I\text{LUN}} = \frac{W}{100 f \cdot b \cdot d \cdot n(10+b)} + \frac{W}{2000n}$$

其中 $W$ 为车辆重量（磅力），$b$ 为轮胎宽度（英寸），$d$ 为轮胎直径（英寸），$n$ 为车轴数量，$f$ 为车轴载荷修正系数。

履带车辆VCI：

$$\text{VCI}_{\text{LV}} = 7.0 + 0.2 M_{I\text{LV}} - \frac{39.2}{M_{I\text{LV}} + 5.6}$$

履带车辆机动指数：

$$M_{I\text{LV}} = \frac{50fW}{b^2 l} + \frac{W}{10 n_t b p}$$

其中 $b$ 为履带宽度（英寸），$l$ 为履带接地长度（英寸），$n_t$ 为负重轮总数，$p$ 为履带节距（英寸）。

通行能力判定准则：

$$\text{若 RCI} > \text{VCI} \Rightarrow \text{可通过}$$

$$\text{若 RCI} \leq \text{VCI} \Rightarrow \text{不可通过}$$

其中 $\text{VCI}_1$ 为1次通过所需最小RCI，$\text{VCI}_{50}$ 为50次通过所需最小RCI。单位换算：$1 \text{ lbf/in}^2 \approx 6.894 \text{ kN/m}^2$。

2.5 水系障碍因子的提取与量化

水系障碍包括河流、湖泊、沼泽等水体，是越野通行中的重要障碍。水系数据可通过OpenStreetMap等开放数据源获取，也可基于遥感影像进行水体提取。

2.5.1 归一化差异水体指数（NDWI）

常用的水体提取方法包括归一化差异水体指数（NDWI）：

$$\text{NDWI} = \frac{\text{GREEN} - \text{NIR}}{\text{GREEN} + \text{NIR}}$$

其中 GREEN 和 NIR 分别为绿光和近红外波段的反射率。当 NDWI > 0.3 时判定为水体或高度饱和区域，RCI取零值（Pundir & Garg, 2022）。

水深和流速是影响可涉水性的关键参数。一般而言，轮式车辆的最大涉水深度为0.5-1.0米，履带式车辆可达1.0-1.5米。流速超过1.5 m/s时，车辆的稳定性将受到显著影响。可涉水判定标准通常综合考虑水深、流速、河床材质等因素，当水深小于车辆最大涉水深度的80%且流速低于1.0 m/s时，可判定为可涉水。

2.6 气象因素的提取与量化

气象因素通过影响土壤含水量、地表摩擦系数等间接影响通行能力。降雨是最主要的动态影响因素，降雨强度与土壤含水量呈正相关，土壤含水量每增加5%，RCI值约降低15%-25%。积雪覆盖可降低地表摩擦系数，积雪深度超过30厘米时，轮式车辆的通行能力将显著下降。冻土状态则直接影响土壤的承载力，冻结土壤的承载力通常是融化状态的3-5倍。

2.6.1 Green-Ampt入渗模型

赵德全（2025）采用Green-Ampt入渗模型描述降雨条件下土壤水分入渗过程：

累积入渗量：

$$F = (K_s - K_i) \cdot t = K_s(\theta_s - \theta_i)(H_f + L)$$

入渗率：

$$i(t) = K_s \left(\frac{\theta_s - \theta_i}{F(t)} + 1\right)$$

湿润锋深度：

$$L(t) = \frac{F(t)}{\theta_s - \theta_i}$$

其中 $K_s$ 为饱和导水率，$K_i$ 为初始导水率，$\theta_s$ 为饱和含水率，$\theta_i$ 为初始含水率，$H_f$ 为湿润锋处吸力水头。

基于Green-Ampt模型可计算PI参数，用于量化降雨条件下车辆通过能力：

$$\text{PI} = \frac{F(t)}{\theta_s(t)}$$

气象数据的时空分辨率对通行能力评估精度至关重要。全球气象再分析数据（如ERA5）的空间分辨率为0.25°×0.25°，时间分辨率为1小时，可满足大尺度分析需求。区域高分辨率气象数据（如CMORPH）的空间分辨率可达8公里，时间分辨率30分钟，更适合精细尺度评估。

2.7 Zevenbergen偏四次曲面系数完整推导

Zevenbergen & Thorne (1987) 提出的偏四次多项式曲面拟合方法已在2.1节给出基本公式。此处补充其完整的系数推导过程，从曲面方程出发，逐步展示如何由3×3高程矩阵的9个点确定9个未知系数。

2.7.1 曲面方程与约束条件

在以中心点 $Z_5$ 为原点的局部坐标系下，3×3窗口内9个点的坐标分别为：

位置编号坐标 $(X, Y)$高程值
1（左上）$(-L, -L)$$Z_1$
2（上中）$(0, -L)$$Z_2$
3（右上）$(L, -L)$$Z_3$
4（左中）$(-L, 0)$$Z_4$
5（中心）$(0, 0)$$Z_5$
6（右中）$(L, 0)$$Z_6$
7（左下）$(-L, L)$$Z_7$
8（下中）$(0, L)$$Z_8$
9（右下）$(L, L)$$Z_9$

偏四次多项式曲面方程为：

$$Z = AX^2Y^2 + BX^2Y + CXY^2 + DX^2 + EY^2 + FXY + GX + HY + I$$

将9个点的坐标代入曲面方程，得到9个约束方程。以下逐步求解每个系数。

2.7.2 系数I的推导（中心点）

将中心点 $(0, 0)$ 代入曲面方程，所有含 $X$ 或 $Y$ 的项均为零：

$$Z_5 = A \cdot 0 + B \cdot 0 + C \cdot 0 + D \cdot 0 + E \cdot 0 + F \cdot 0 + G \cdot 0 + H \cdot 0 + I = I$$

因此：

$$I = Z_5$$

2.7.3 系数G和H的推导（一阶偏导数）

系数G：取 $Y=0$ 时的三个点 $Z_4(-L,0)$, $Z_5(0,0)$, $Z_6(L,0)$，此时曲面方程简化为 $Z = DX^2 + GX + I$：

$$Z_4 = DL^2 - GL + Z_5 \quad \cdots (i)$$

$$Z_6 = DL^2 + GL + Z_5 \quad \cdots (ii)$$

式 (ii) 减式 (i)：

$$Z_6 - Z_4 = 2GL \implies G = \frac{Z_6 - Z_4}{2L}$$

系数H：取 $X=0$ 时的三个点 $Z_2(0,-L)$, $Z_5(0,0)$, $Z_8(0,L)$，曲面简化为 $Z = EY^2 + HY + I$：

$$Z_2 = EL^2 - HL + Z_5 \quad \cdots (iii)$$

$$Z_8 = EL^2 + HL + Z_5 \quad \cdots (iv)$$

式 (iv) 减式 (iii)：

$$Z_8 - Z_2 = 2HL \implies H = \frac{Z_8 - Z_2}{2L}$$

2.7.4 系数D和E的推导（二阶纯偏导数）

系数D：式 (i) 加式 (ii)：

$$Z_4 + Z_6 = 2DL^2 + 2Z_5 \implies D = \frac{Z_4 + Z_6 - 2Z_5}{2L^2}$$

系数E：式 (iii) 加式 (iv)：

$$Z_2 + Z_8 = 2EL^2 + 2Z_5 \implies E = \frac{Z_2 + Z_8 - 2Z_5}{2L^2}$$

2.7.5 系数F的推导（二阶混合偏导数）

将四个角点代入曲面方程，利用对称性消除其他系数。以 $Z_1(-L,-L)$ 为例展开：

$$Z_1 = A L^4 - B L^3 - C L^3 + D L^2 + E L^2 + F L^2 - GL - HL + Z_5$$

类似地写出 $Z_3(L,-L)$, $Z_7(-L,L)$, $Z_9(L,L)$：

$$Z_3 = A L^4 + B L^3 - C L^3 + D L^2 + E L^2 - F L^2 + GL - HL + Z_5$$

$$Z_7 = A L^4 - B L^3 + C L^3 + D L^2 + E L^2 - F L^2 - GL + HL + Z_5$$

$$Z_9 = A L^4 + B L^3 + C L^3 + D L^2 + E L^2 + F L^2 + GL + HL + Z_5$$

构造组合 $(Z_1 - Z_3 + Z_9 - Z_7)$：展开后 $A, B, C, D, E, G, H$ 的各项因对称性相互抵消，仅保留 $F$ 项：

$$Z_1 - Z_3 + Z_9 - Z_7 = 4FL^2$$

因此：

$$F = \frac{Z_1 - Z_3 + Z_9 - Z_7}{4L^2}$$

2.7.6 系数A、B、C的推导

系数A：四个角点之和：

$$Z_1 + Z_3 + Z_7 + Z_9 = 4AL^4 + 2(D+E)L^2 + 4Z_5$$

代入已求得的 $D$ 和 $E$，化简得：

$$A = \frac{(Z_1 + Z_3 + Z_7 + Z_9) - 2(Z_2 + Z_4 + Z_6 + Z_8) + 4Z_5}{4L^4}$$

系数B：利用上行两角与下行两角之差的组合：

$$B = \frac{(Z_1 + Z_3 - Z_7 - Z_9) - 2(Z_2 - Z_8)}{4L^3}$$

系数C：利用左列两角与右列两角之差的组合：

$$C = \frac{(-Z_1 + Z_3 - Z_7 + Z_9) - 2(Z_4 - Z_6)}{4L^3}$$

2.7.7 地形参数的解析计算

对曲面方程求偏导数，在原点 $(X=Y=0)$ 处：

$$\frac{\partial Z}{\partial X}\bigg|_0 = G, \quad \frac{\partial Z}{\partial Y}\bigg|_0 = H$$

二阶偏导数：

$$\frac{\partial^2 Z}{\partial X^2}\bigg|_0 = 2D, \quad \frac{\partial^2 Z}{\partial Y^2}\bigg|_0 = 2E, \quad \frac{\partial^2 Z}{\partial X \partial Y}\bigg|_0 = F$$

沿任意方向 $\phi$ 的曲率为二阶方向导数：

$$\text{Curvature}(\phi) = -2(D\cos^2\phi + E\sin^2\phi + F\cos\phi\sin\phi)$$

因此：

坡度：$\text{SLOPE} = \sqrt{G^2 + H^2}$

坡向：$\theta = \arctan(-H / -G)$（分子分母符号决定象限）

剖面曲率（$\phi = \theta$）：

$$\text{PROFC} = \frac{-2(DG^2 + EH^2 + FGH)}{G^2 + H^2}$$

平面曲率（$\phi = \theta + \pi/2$）：

$$\text{PLANC} = \frac{2(DH^2 + EG^2 - FGH)}{G^2 + H^2}$$

2.7.8 汇水面积计算与Evans方法比较

汇水面积：基于D8算法思想，每个格网点代表一个面积单元 $L^2$，将其汇水面积传递给坡向指向的下游邻域格网。程序从四个方向反复扫描矩阵，直到所有格网点的汇水面积确定。

2.8 Suvinen车辆力学模型完整公式链

Suvinen (2006) 建立了从车辆参数到通行代价面的完整物理力学模型链。该模型以面向对象的方法将车辆、地形、天气等因素描述为独立对象，通过物理力学公式将各阻力分量统一为力（kN）的量纲后直接相加，不使用主观权重——各阻力来自物理定律自然综合。

2.8.1 车辆轮载分配模型

车辆轮载分配是整个力学模型链的起点。假设前轴承受总质量的60%，后轴承受总质量的40%加上载荷质量：

$$W_f = \frac{0.6 \times M_T}{n_W} \times g \tag{Suvinen.1}$$

$$W_r = \frac{(0.4 \times M_T + M_L)}{n_W} \times g \tag{Suvinen.2}$$

其中 $M_T$ 为拖拉机总质量(kg)，$M_L$ 为载荷质量(kg)，$n_W$ 为轮轴上的车轮数，$g = 9.81 \text{ m/s}^2$。典型参数：总质量15,000 kg，最大载荷14,000 kg，轮胎8个。

2.8.2 轮胎变形与接触压力

轮胎变形量采用Saarilahti模型：

$$\delta = \frac{0.365 \times 170 \times W_i}{p_i \times 1000}$$

其中 $p_i$ 为充气压力(kPa)，$W_i$ 为轮载(kN)。

轮胎接地接触压力采用Maclaurin公式：

$$q = \frac{W_i}{b^{0.8} \times d^{0.8} \times d^{0.4}}$$

其中 $b$ 为轮宽(m)，$d$ 为轮径(m)。

2.8.3 土壤承载力模型（塑性理论）

安全轮载采用Silversides & Sundberg公式（基于塑性极限分析）：

$$W_{i,\max} = \pi \times r_{\text{theor}}^2 \times (1.3 \times c \times N_C + 0.6 \times \gamma \times r_{\text{theor}} \times N_c)$$

承载力因子 $N_c$ 采用Balla拟合公式：

$$N_c = 0.0488\phi^3 - 3.6055\phi^2 + 90.9482\phi + 760.7648$$

其中 $r_{\text{theor}}$ 为理论圆形接触面积半径(m)，$c$ 为粘聚力(kN/m²)，$\gamma$ 为单位重量(kN/m³)，$\phi$ 为内摩擦角(°)。

矩形接触面积的Skempton承载力因子：

$$N_C = 5 \times \left(1 + 0.2 \times \frac{B^*}{L^*}\right)$$

2.8.4 滚动阻力模型

单轮滚动阻力：

$$F_{Ri} = \lambda_R \times W_i$$

滚动阻力系数（刚性轮虚拟轮模型）：

$$\lambda_R = 0.04 + \frac{z}{\sqrt{2R \times z}}$$

其中 $0.04$ 为轮胎内滚动阻力，$z$ 为沉降量(m)，$R$ 为虚拟轮半径(m)。

沉降量采用Janbu公式：

$$z = \frac{q \times b}{E}$$

其中 $E$ 为杨氏模量(kN/m²)。

2.8.5 坡度阻力模型

$$F_G = \lambda_G \times W = \sin\alpha \times W$$

侧倾角限制：

$$\alpha_{l,\max} = \frac{8.5}{\cos(90° - \chi)}$$

其中 $\chi$ 为行驶方向与坡面方位的夹角。下坡非线性处理规则：下坡 < 5°为助力（负阻力），5°~10°助力递减，> 10°转为阻力，> 15°等同于 -5° 上坡阻力。

2.8.6 障碍物阻力与雪阻模型

障碍物阻力系数（Saarilahti模型）：

$$\lambda_O = \frac{\sum_{i=1}^{P_i} k \times H_i}{D}$$

其中 $k = 0.3$ 为能量损失因子，$H_i$ 为障碍物高度(m)，$D$ 为行驶距离(m)。

雪阻模型：

$$F_S = 35 \times h_S \quad (\text{低密度雪})$$

$$F_S = 60 \times h_S \quad (\text{高密度雪})$$

2.8.7 冻土效应模型

冻深采用Stefan公式：

$$z_{\text{frost}} = s \times \sqrt{a \times F^-}$$

其中 $s$ 为土壤类型系数，$F^-$ 为零下温度小时数(°C·h)。

冻土扩散后的接触面积增大：

$$B^{*'} = B^* + 2 \times z_{\text{frost}} \times \tan\beta$$

$$L^{*'} = L^* + 2 \times z_{\text{frost}} \times \tan\beta$$

冻土承载力因子（Onninen方法，将冻土视为"板"扩散压力）：

$$N_C' = 5 \times \left(1 + 0.2 \times \frac{B^{*'}}{L^{*'}}\right) \times \left(1 + 0.2 \times \frac{z_{\text{frost}}}{B^{*'}}\right)$$

冻土效应可使承载力提高5-10倍。解冻深度采用Luoma公式：$z_{\text{thaw}} = 1.1 \times \sqrt{F^+}$。

2.8.8 综合阻力与路径规划代价面

物理加法综合模型：

$$F_{\text{total}} = F_R + F_S + F_G + F_O$$

所有力以kN为单位直接相加，各阻力来自物理定律自然综合，无需主观赋权。

路径规划采用ArcGIS PATHDISTANCE公式：

$$\text{Cost} = S_d \times V_f \times \frac{(F_a + H_a)/2 + (F_b + H_b)/2}{2}$$

其中 $S_d$ 为表面距离，$V_f$ 为垂直因子，$F$ 为摩擦力，$H$ 为水平因子。代价面建立后，使用Dijkstra或A*算法搜索最小代价路径。

2.8.9 完整计算流程

$$\text{GIS数据} \to \text{轮载分配} \to \text{接触压力} \to \text{承载力检查} \to \text{沉陷量} \to \text{滚动阻力}$$

$$+ \text{坡度阻力} + \text{障碍物阻力} + \text{雪阻} \to \text{总阻力(代价面)} \to \text{路径规划}$$

2.9 张文诗综合速度模型

张文诗提出了从地形因素到车辆机动速度的完整建模框架，涵盖沿道路通行和越野通行两个维度，其核心特征是基于实验数据的系数标定。

2.9.1 沿道路综合机动速度模型

坦克综合机动速度的乘法模型：

$$v = v_0 \cdot K_1 \cdot K_2 \cdot K_3 \cdot K_4 \cdot K_5 \cdot K_6 \cdot K_7$$

其中各系数定义如下：

系数影响因素典型取值
$K_1$路面材料沥青/水泥=1.0, 砂石=0.6~0.8, 土质=0.4~0.6
$K_2$路面损坏程度轻损=0.75, 重损=0.5
$K_3$纵坡度三次样条拟合（实验数据分段建立）
$K_4$路面宽度与车辆宽度比相关
$K_5$降雨影响与降雨强度相关
$K_6$降雪影响与积雪深度相关
$K_7$雾天影响与能见度相关

该模型的核心思想是乘法综合——各因素独立影响基准速度，任一因素极端不利时速度趋近于零，符合越野通行的"短板效应"。

2.9.2 纵坡度影响系数的三次样条拟合

纵坡度影响系数 $K_3$ 不采用简单的线性关系，而是根据实验数据分段建立三次样条函数：

$$K_3(\theta) = a_i + b_i(\theta - \theta_i) + c_i(\theta - \theta_i)^2 + d_i(\theta - \theta_i)^3$$

其中 $\theta_i$ 为第 $i$ 段的起始坡度，$a_i, b_i, c_i, d_i$ 为该段的样条系数。分段点处保证函数值和一阶、二阶导数连续，使整体曲线光滑。

2.9.3 越野通行的有效通行面积比

越野通行性能采用有效通行面积比衡量：

$$S_{\text{有效}} = S_1 + \sum K_j \cdot S_j$$

多通行主体的有效通行面积比：

$$S_{\text{多}} = \sum \lambda_i \cdot S_{\text{有效},i}$$

其中 $\lambda_i$ 为第 $i$ 种通行主体的权重，$S_1$ 为完全可通行面积，$K_j$ 为第 $j$ 类区域的通行系数。

2.9.4 路段综合权值与最佳路径

路段综合权值采用隶属度加权：

$$P' = P_a \cdot \mu_a + P_t \cdot \mu_t + P_c \cdot \mu_c$$

其中 $\mu_a = 1 - l/l_0$（距离隶属度，$l_0$ 为平均路段长度），$\mu_t = v'/v_{\max}$（时间隶属度），$\mu_c$ 为隐蔽条件隶属度。

最佳路径用Dijkstra标号法求解，取最大综合权值为标号。

2.10 黄鲁峰GIS因子综合分析方法

黄鲁峰提出了基于GIS的战场自然环境因子综合分析框架，将地形、气象、水文、土质、植被等多因子统一到地理格网模型中，并采用模糊综合评判方法进行定量化评估。

2.10.1 基于地理格网的多因子数据模型

将目标区域按格网粒度划分成规则网格单元，每个单元包含地形、水文、气象、土质、植被等全部环境因子值。这种格网模型的核心优势在于：

  • 统一存储：每个格网单元是一个完整的环境描述，不同类型因子可在同一空间框架下管理和计算
  • 叠置分析：支持合成叠置（多因子运算生成新属性）和统计叠置（分析因子空间分布特征）
  • 粒度可调：格网粒度可根据分析需求灵活调整，宏观分析用大粒度，精细分析用小粒度

2.10.2 理想状态通行性分析

基于车辆动力平衡方程，推导车辆在给定地形条件下的最大行驶速度：

$$N \cdot \mu_h \cdot \eta = (f \cdot G\cos\alpha + G\sin\alpha + k \cdot S \cdot v_w^2) \cdot v$$

其中 $N$ 为发动机功率，$\mu_h$ 为气压影响系数，$\eta$ 为机械效率，$f$ 为摩擦系数，$G$ 为车重，$\alpha$ 为坡度角，$k$ 为空气阻力系数，$S$ 为正面投影面积，$v_w$ 为逆风分量。

有效通行面积比：

$$R = \frac{\sum A_i(\alpha < \alpha_{\text{临界}}) + \sum C_j \cdot A_j}{\sum A_{\text{总}}}$$

其中 $\alpha_{\text{临界}}$ 为车辆最大爬坡角，$C_j$ 为第 $j$ 类区域的通行系数。

2.10.3 模糊综合评判方法

对于定性因子（如植被类型、土质类型）的量化评估，采用模糊综合评判方法：

设评判对象因子集 $U = \{u_1, u_2, ..., u_n\}$，评判等级集 $V = \{v_1, v_2, ..., v_m\}$，通过隶属函数建立模糊评判矩阵 $R$，结合权重向量 $A$，合成得：

$$B = A \circ R$$

归一化后确定评判等级。权重确定采用两种方法：

  • 分类权重：采用投影寻踪法确定
  • 排序权重：采用特征根法确定

2.10.4 多要素叠置分析

多要素叠置分析分为两个层次：

  • 合成叠置：对多个因子图层进行代数运算，生成新的综合属性图层（如坡度×土壤承载力→综合通行性）
  • 统计叠置：分析各因子在空间上的分布规律，识别关键限制因子

该方法的创新之处在于将模糊数学引入战场环境分析，解决了多因子综合作用下的定量化评估问题，同时采用组件式GIS开发架构（Composite和Bridge设计模式），实现了分析模型与界面的解耦。

2.12 批判性思考与改进方向

当前单要素量化建模存在以下关键问题：

因素间的交互作用：各通行阻碍因子并非独立作用，而是存在复杂的交互效应。例如，降雨不仅直接影响土壤含水量，还会改变地表粗糙度和植被覆盖状态。现有的单要素模型难以准确捕捉这些交互作用，可能导致综合评估结果的偏差。

动态更新需求：土壤含水量、积雪覆盖、植被状态等因子具有显著的时间变异性，需要建立动态更新机制。现有的静态模型难以反映这些因子的实时变化，建议结合物联网传感器和实时遥感数据，构建动态更新的通行能力评估系统。

车辆适应性差异：不同车辆类型（轮式/履带式、轻型/重型）对各因子的敏感性存在显著差异。例如，履带式车辆对坡度的适应性更强，但对土壤承载力的要求更高。现有的通用模型难以准确反映这种车辆特异性，建议建立车辆-因子交互矩阵，实现车辆适应性的精细化评估。

尺度效应：不同空间尺度下，各因子的影响权重和量化方法可能存在差异。例如，在公里尺度上，坡度可能是主导因子；而在米尺度上，地表粗糙度和微地形的影响更为显著。需要建立多尺度评估框架，实现从宏观到微观的精细化分析。

综上所述，单要素量化建模为越野通行能力评估提供了基础方法，但在交互作用、动态性、车辆适应性和尺度效应等方面仍需进一步改进。这些改进方向将为第三章的多因子综合评估提供重要参考。

3. 综合通行能力的多因子评估模型与方法

单要素阻碍度模型为通行能力评估提供了基础，但实际越野环境中各因素相互交织、共同作用，需要建立科学的多因子综合评估模型。本章系统阐述综合通行能力评估模型的构建方法、权重确定技术、模型验证与比较，以及典型应用场景，并通过案例分析展示模型的实际应用效果。

3.1 多因子综合评估模型的基本框架

多因子综合评估的核心思想是将多个单要素阻碍度通过加权融合为统一的综合通行能力指标。当前主流的综合评估模型可分为以下几类：

3.1.1 加权线性模型

Liu等人（2025）构建的动态通行能力图采用经典的加权线性模型Liu et al. (2025)：

$$P_i = W_1 P_1 + W_2 P_2 + \cdots + W_{11} P_{11}$$

其中 $P_i$ 为第 $i$ 个栅格内车辆的综合通行性值，$W_1 \sim W_{11}$ 为11个环境因素通过层次分析法（AHP）确定的权重系数，$P_1 \sim P_{11}$ 为各环境因素基于参考比较经验阈值法（RCTM）量化的单因素通行性值。

该模型的特点在于：首先，采用加权融合方式，能够灵活调整各因素的相对重要性；其次，通过动态因素（如气象、土壤含水量）的引入，实现通行能力图的时间更新；最后，在最终地图构建中，将各因子不可通行区域的并集从初步地图中剔除，确保安全性Liu et al. (2025)。

3.1.2 综合速度与通行时间公式

Xu et al. (2022) 提出了基于单因子速度系数的综合通行速度计算方法。单因子通行速度为：

$$V_x = V_{\text{Max}} \times f_v$$

综合速度系数（加权平均）为：

$$f_v^t = \frac{W_V \cdot f_v^V + W_W \cdot f_v^W + W_G \cdot f_v^G + W_{DEM} \cdot f_v^{DEM} + W_D \cdot f_v^D}{W_V + W_W + W_G + W_{DEM} + W_D}$$

综合通行速度为：

$$V = V_{\text{Max}} \times f_v^t$$

相邻栅格的通行时间计算公式为：

直行距离：

$$T = \frac{R(V + V_1)}{2VV_1}$$

斜向距离：

$$T' = \frac{\sqrt{2} R (V + V_1)}{2VV_1}$$

其中 $R$ 为栅格边长，$V$ 为当前栅格速度，$V_1$ 为前方栅格速度。

3.1.4 其他综合评估方法

TOPSIS方法（Technique for Order Preference by Similarity to Ideal Solution）：基于各方案与正理想解和负理想解的距离进行排序，适用于多属性决策问题。该方法通过计算各评估单元与理想解的相对接近度来确定通行能力等级，在道路选线等领域已有应用Highway Route Planning via MCDA Integration (2021)。

VIKOR方法（VIseKriterijumska Optimizacija I Kompromisno Resenje）：基于折中排序的方法，通过最大化群体效用和最小化个体遗憾来确定最优方案。VIKOR在处理冲突性指标时具有优势，特别适用于需要平衡多个相互矛盾因素的通行能力评估场景Highway Route Planning via MCDA Integration (2021)。

机器学习方法：2026年的一项研究提出了局部到全局多尺度加速机动地图预测方法（LGMM），通过深度学习自动学习各因素的权重和非线性关系，避免了人工设定权重的主观性LGMM (2026)。机器学习方法能够捕捉因素间的复杂交互作用，但需要大量标注数据进行训练。

然而，线性加权模型假设各因素对通行能力的影响是可加的，忽略了因素间的交互作用。有学者认为，在复杂地形中，坡度与土壤类型的交互影响可能比单独影响更为显著基于复杂地表环境与车辆受力分析的越野机动路径规划方法。

3.2 权重确定技术

权重确定是多因子评估模型的核心环节，直接影响评估结果的可靠性。当前主流方法包括：

3.2.1 层次分析法（AHP）完整计算流程

层次分析法（AHP）由Saaty于1980年提出，是一种系统化的多准则决策方法Saaty (1980) The Analytic Hierarchy Process。以下以Xu et al. (2022)的5因子AHP实例展示完整计算流程。

步骤1：构建判断矩阵

采用1-9标度法，通过专家评分和数据调查，对影响通行效率的5个因子（植被、水系、地质、灾害、DEM）进行两两比较，构建判断矩阵 $A$：

$$A = \begin{pmatrix}

1 & 1/2 & 1/4 & 1/2 & 1/3 \\

2 & 1 & 1/3 & 1/3 & 1/2 \\

4 & 3 & 1 & 1/2 & 2 \\

2 & 3 & 2 & 1 & 3 \\

3 & 2 & 1/2 & 1/3 & 1

\end{pmatrix}$$

其中，矩阵元素 $a_{ij}$ 表示因子 $i$ 相对于因子 $j$ 的重要程度。

步骤2：计算特征向量（权重向量）

对判断矩阵 $A$ 求解最大特征值对应的特征向量，得到各因子的权重：

$$W = [W_{\text{vegetation}}, W_{\text{water}}, W_{\text{geology}}, W_{\text{disaster}}, W_{DEM}] = [0.087, 0.110, 0.275, 0.357, 0.171]$$

步骤3：计算最大特征值 $\lambda_{\max}$

$$\lambda_{\max} = \sum_{i=1}^{n} \frac{[AW]_i}{n w_i}$$

计算结果：$\lambda_{\max} = 5.292$

步骤4：计算一致性指标 CI

$$CI = \frac{\lambda_{\max} - n}{n - 1}$$

其中 $n$ 为因子数量（此处 $n=5$）。计算结果：$CI = \frac{5.292 - 5}{5 - 1} = 0.073$

步骤5：计算一致性比率 CR

$$CR = \frac{CI}{RI}$$

其中 $RI$ 为随机一致性指标，取决于矩阵阶数：

阶数12345678910
$RI$000.580.901.121.241.321.411.451.49

计算结果：$CR = \frac{0.073}{1.12} = 0.065$

步骤6：一致性检验判定

若 $CR < 0.10$，则判断矩阵通过一致性检验。本例 $CR = 0.065 < 0.10$，通过一致性检验。

Liu等人（2025）采用AHP确定11个因素的权重，要求一致性比率 $CR < 0.1$Liu et al. (2025)。AHP的优势在于能够融合专家知识，通过两两比较构建判断矩阵，但主观性较强，不同专家可能给出差异较大的权重。

3.2.4 其他权重确定方法

熵权法：基于各因素数据的信息熵确定权重，数据离散程度越大，权重越高。该方法客观性强，但可能忽略因素的实际重要性。熵权法的典型应用包括：在土地适宜性评价中，熵权法被用于确定评价指标的客观权重，与AHP结合形成组合权重以平衡主观性与客观性Off-road trafficability using MCDA (2026)。熵权法计算简单，适用于数据量较大的场景，但对异常值较为敏感。

TOPSIS/VIKOR方法：这些方法不仅可用于综合评估，还可用于权重确定。TOPSIS通过计算各指标与理想解的距离来反映指标的重要性；VIKOR通过群体效用和个体遗憾的平衡来确定权重分配。2026年的一项MCDA综述研究对比了包括TOPSIS、VIKOR在内的多种多准则决策分析方法，指出不同方法在不同应用场景下各有优劣Off-road trafficability using MCDA (2026)。

模糊综合评估方法：基于Zadeh于1965年提出的模糊集合理论Zadeh (1965) Fuzzy Sets，通过隶属函数和模糊运算处理不确定性和模糊性。模糊AHP方法结合了AHP和模糊理论的优点，能够处理判断矩阵中的不确定性Research on the System and Method of Off-road Vehicle Passability (2024)。模糊综合评估的典型应用包括：在土壤侵蚀风险评估中，模糊综合评估方法被用于处理土壤类型、坡度等因素的不确定性；在军事机动性评估中，模糊方法被用于处理战场环境信息的模糊性和不确定性。

组合赋权法：将主观权重（如AHP）与客观权重（如熵权法）通过线性组合或乘法合成得到综合权重。2026年的一项研究采用多标准决策分析方法，结合GIS空间分析构建车辆通行性地图，实现了主客观权重的有效融合Off-road trafficability using MCDA (2026)。组合赋权法能够兼顾专家经验和数据客观性，是当前研究的热点方向。

机器学习方法：2026年的一项研究提出了局部到全局多尺度加速机动地图预测方法（LGMM），通过深度学习自动学习各因素的权重，避免了人工设定权重的主观性LGMM (2026)。

权重确定的挑战在于：不同地区、不同车辆类型的最佳权重可能不同；因素数量增加时，AHP的判断矩阵规模急剧增大；动态因素的权重需要随时间调整。

3.3 模型验证与比较

模型验证是评估综合通行能力模型可靠性的关键环节。当前主要验证方法包括：

精度评估指标：2026年的一项研究通过综合实地调查和统计指标进行精度评估，总体精度达到84.7%，Kappa系数为0.81Off-road trafficability using MCDA (2026)。这两个指标分别衡量了模型的整体分类准确性和分类结果与随机分类的差异。

路径规划验证：通过将综合通行能力图用于路径规划，验证规划路径的合理性。Liu等人（2025）采用改进A*算法进行路径规划，验证了动态通行能力图的有效性Liu et al. (2025)。

实地试验验证：武汉大学的研究通过实装试验验证车辆车速估计精度优于90%，车辙深度误差优于2cm无道路环境车辆支承通过性全遥感反演方法。

交叉验证与敏感性分析：使用独立数据集验证模型的泛化能力，并评估各参数对模型输出的影响程度。具体方法包括：k折交叉验证（将数据集分为k个子集，轮流使用k-1个子集训练、1个子集验证）；留一法交叉验证（每次留出一个样本作为验证集）；参数扰动分析（对权重参数进行±10%~±20%的扰动，观察评估结果的变化幅度）。

模型比较方面，需要对比不同权重确定方法、不同因素组合、不同融合策略的评估效果。然而，当前越野通行能力评估领域缺乏统一的比较基准和标准化的评估流程，这已成为阻碍不同研究方法公平比较和领域发展的主要障碍Verti-Bench (2025)。

因素选择：基于文献分析和专家咨询，选择8个关键因素：

  1. 坡度（Slope）
  2. 地形位置指数（TPI）
  3. 地表粗糙度（Roughness）
  4. 土壤锥指数（Cone Index）

  • 水系障碍：0.07

基于权重向量和标准化因素值，计算每个栅格的综合通行能力值。模型计算结果显示，综合通行能力值在0.12-0.95之间，平均值为0.62，标准差为0.18。将综合通行能力值分为三个等级：

  • 可通行（>0.7）：占区域面积的45%

3.4.6 案例的局限性和推广条件

  4. 车辆类型单一：仅针对轮式越野车辆，其他车辆类型需要重新标定

推广条件：

  1. 地形适用性：适用于山地、丘陵、平原等多种地形，但需要根据具体地形调整因素选择
  2. 数据要求：需要30米或更高分辨率的DEM数据，土壤数据分辨率越高越好
  3. 车辆适应性：不同车辆类型需要重新确定权重和阈值
  4. 区域调整：不同地区的因素权重可能需要重新标定

军事机动性评估：北约参考机动性模型（NRMM）及其下一代（NG-NRMM）是国际标准，用于评估军事车辆在不同地形条件下的通行能力NG-NRMM (2024)。该模型考虑了地形、土壤、植被等多种因素，为军事行动规划提供支持。

自动驾驶越野导航：随着自动驾驶技术的发展，越野环境下的自主导航需求日益增长。2025年的一项综述系统总结了越野无人驾驶技术，包括感知、规划、控制等方面越野环境下无人驾驶车辆技术研究综述。

农业林业作业：在农田、林地中进行机械化作业时，需要评估土壤承载力以避免设备陷入。2025年的一项研究提出了基于遥感的车辆支承通过性全链条反演方法无道路环境车辆支承通过性全遥感反演方法。

3.6 模型的局限性与改进方向

局限性1：因素选择的主观性。当前研究在因素选择上缺乏统一标准，不同研究考虑的因素数量和类型差异较大。有学者认为，因素选择应基于因素对通行能力影响的显著性分析A comprehensive approach for off-road trafficability evaluation (2022)。

局限性2：权重的区域适用性。在某一地区确定的权重可能不适用于其他地区，需要建立区域自适应的权重确定机制。

局限性3：动态更新的实时性。虽然Liu等人（2025）提出了动态通行能力图的概念Liu et al. (2025)，但气象、土壤等动态数据的实时获取仍是挑战。

局限性4：车辆类型的泛化。当前模型多针对特定车型，缺乏通用的车辆适应性框架。

局限性5：数据质量与计算效率的权衡。高分辨率数据能够提高模型精度，但计算成本急剧增加。研究表明，30米分辨率DEM数据在精度和计算效率之间取得了较好平衡NASADEM-30米精度全球DEM数据。然而，对于精细尺度的车辆动力学分析，可能需要更高分辨率数据。

改进方向：

  1. 建立标准化的因素选择框架
  2. 发展区域自适应权重方法
  3. 实现真正的实时更新
  4. 构建车辆-环境耦合模型
  5. 优化数据分辨率与计算效率的平衡

3.7 多尺度评估策略

实际应用中，通行能力评估需要在不同尺度上进行。2026年LGMM研究提出了局部到全局多尺度加速机动地图预测方法LGMM (2026)：

局部尺度：利用高分辨率数据（如无人机影像）进行精细评估，适用于小范围关键区域。

区域尺度：利用中分辨率数据（如30m DEM）进行区域评估，适用于大范围路径规划。

全局尺度：利用低分辨率数据进行快速筛选，识别潜在可通行区域。

多尺度策略的优势在于：在保证精度的同时降低计算成本；能够根据应用需求灵活调整评估粒度。

车辆参数获取的现实挑战：不同尺度评估对车辆参数的要求不同。局部尺度需要详细的车辆几何和动力学参数，而全局尺度可能只需要车辆类型分类。数据显示，车辆参数的获取存在现实挑战：厂家数据可能不完整，实地测量成本高，遥感反演精度有限无道路环境车辆支承通过性全遥感反演方法。因此，需要建立车辆参数数据库和标准化获取方法。

3.8 综合评估模型的发展趋势

基于对现有文献的分析，综合通行能力评估模型呈现以下发展趋势：

趋势1：从静态到动态。随着气象、土壤等动态数据获取能力的提升，通行能力评估正从静态向动态发展。Liu等人（2025）的动态通行能力图是这一趋势的典型代表Liu et al. (2025)。

趋势2：从经验到物理。传统模型多基于经验数据，未来将更多融入物理机制。武汉大学的全遥感反演方法将土壤力学原理与机器学习相结合，代表了这一方向无道路环境车辆支承通过性全遥感反演方法。

趋势3：从单模态到多模态。2025年的一项研究提出了跨模态特征融合网络，结合遥感影像和不完整辅助数据进行通行性评估Off-Road Trafficability Assessment with Cross-Modal Fusion (2025)。

趋势4：从离线到在线。随着边缘计算和物联网技术的发展，通行能力评估将实现实时在线更新。

3.9 批判性思考与改进方向

3.9.1 现有方法的系统性局限

权重确定的主观性问题是核心挑战之一。AHP作为最常用的主观权重确定方法，其判断矩阵依赖专家经验，不同专家可能给出不同判断，导致权重结果具有主观性。研究表明，AHP的一致性比率 $CR < 0.1$ 为可接受标准，但这一标准本身具有一定主观性Liu et al. (2025)。有研究对比了AHP与熵权法的实际效果：在道路选线研究中，AHP确定的权重与熵权法确定的权重存在显著差异，AHP更依赖专家经验，而熵权法更依赖数据特征；组合赋权法（如AHP-熵权法组合）能够在一定程度上平衡两者的优势Highway Route Planning via MCDA Integration (2021)。然而，组合权重的最优组合方式仍需深入研究。

静态模型与动态环境的矛盾是另一重要局限。传统多因子评估模型通常基于静态环境参数，但实际越野环境中，土壤含水量、植被覆盖等因素会随时间动态变化。数据显示，考虑气象因素（降雨、降雪、风力）的时空变化可以构建动态通行能力图，但这增加了模型的复杂性和计算成本Liu et al. (2025)。

车辆类型差异化的考虑程度存在不足。不同车辆类型（轮式、履带式、多轴车辆）具有不同的机动特性，但许多研究采用统一的车辆模型，这可能影响评估精度。研究表明，车辆几何参数（轴距、轮距、离地间隙）和动力学参数（发动机特性、传动系统）对通行能力有显著影响，需要针对不同车辆类型建立差异化的评估模型基于复杂地表环境与车辆受力分析的越野机动路径规划方法。

数据精度与模型精度的矛盾是技术实现的关键问题。遥感数据提供了大范围、多时相的环境信息，但其精度可能无法满足精细尺度的车辆动力学分析。数据显示，土壤含水量遥感反演的RMSE为4.9-6.3%，车辆车速预测精度优于90%，这表明遥感数据在精度和覆盖范围之间需要权衡无道路环境车辆支承通过性全遥感反演方法。

3.9.2 数据质量影响分析

数据误差传播分析表明，输入数据的误差会通过模型传播到最终结果。研究发现，土壤锥指数具有显著的空间变异性，这对车辆机动性评估提出了重要挑战A comprehensive approach for off-road trafficability evaluation (2022)。数据显示，土壤锥指数的测量误差可达15-20%，这直接影响土壤承载力评估的准确性。

数据分辨率对模型精度的影响需要仔细考虑。数字高程模型（DEM）的分辨率决定了地形分析的精细程度。研究表明，30米分辨率的DEM数据能够满足大多数越野通行性分析的需求，但对于精细尺度的车辆动力学分析可能不足NASADEM-30米精度全球DEM数据。

数据时效性对评估结果的影响不容忽视。环境因素随时间变化，过时的数据可能导致评估结果不准确。事实表明，土壤含水量、植被覆盖等因素具有明显的季节性和年际变化，需要使用最新的数据进行评估无道路环境车辆支承通过性全遥感反演方法。

3.9.3 计算复杂度讨论

不同权重确定方法的计算复杂度存在显著差异。AHP需要进行 $n(n-1)/2$ 次成对比较（$n$ 为因素数量），判断矩阵的特征值计算涉及 $n$ 阶矩阵运算，计算成本随因素数量增加而显著增长Saaty (1980) The Analytic Hierarchy ProcessCalculating Maximum Eigenvalues in Pairwise Comparison Matrices (2025)。熵权法的计算复杂度相对较低，主要涉及信息熵计算，计算量与样本数量 $m$ 和因素数量 $n$ 的乘积成正比。模糊综合评估方法的计算复杂度较高，涉及隶属函数计算和模糊运算Zadeh (1965) Fuzzy Sets。

大规模应用的计算挑战需要特别关注。当应用范围扩大到大区域时，数据量和计算量急剧增加。数据显示，一个省域范围的越野通行能力评估可能涉及数百万个栅格单元，每个单元需要计算多个因素的权重和综合值。有学者认为，并行计算和分布式计算技术可以缓解大规模应用的计算压力。

3.9.4 改进方向

基于上述分析，未来研究可以在以下方向进行改进：

深度学习方法的应用是重要方向。研究表明，跨模态特征融合网络可以用于越野通行性评估，提高评估精度Off-Road Trafficability Assessment with Cross-Modal Fusion (2025)。有学者认为，深度学习方法能够自动学习特征，减少人工干预，但需要大量标注数据。

动态模型的开发是另一重要方向。事实表明，考虑气象因素的时空变化可以构建动态通行能力图Liu et al. (2025)。未来研究可以结合实时气象数据和土壤含水量监测数据，开发动态更新的通行能力评估模型。

车辆-环境深度耦合是提高评估精度的关键。研究表明，基于物理的建模方法可以更好地模拟车辆-环境交互作用基于复杂地表环境与车辆受力分析的越野机动路径规划方法。未来研究可以结合车辆动力学模型和地面力学模型，实现车辆-环境的深度耦合。

多源数据融合是提高数据质量的有效途径。数据显示，遥感数据、实地测量数据和历史数据各有优缺点，通过数据融合可以取长补短无道路环境车辆支承通过性全遥感反演方法。未来研究可以开发多源数据融合算法，提高数据质量和覆盖范围。

3.10 He (2023) AHP-WIC定量规则法完整推导

He et al. (2023) 提出了AHP-WIC（加权信息含量）定量规则法，将传统定性的人工规则方法转化为定量的信息含量计算。本节完整展示该方法的推导过程。

3.10.1 问题背景：规则数量的指数爆炸

当通行性评估涉及多个因素时，交通规则数量呈指数增长：

$$Q = \prod_{i=1}^{N} S_i$$

其中 $Q$ 为交通规则总数，$N$ 为评估因素数量，$S_i$ 为因素 $i$ 的分类参数数量。本文10个因素（坡度、土地覆盖、土壤、含水量、RCI、地质灾害、TPI、SPI、高程、岩石、节理构造）理论上可生成 $Q = 342,144$ 条规则，人工计算和管理极为困难。

3.10.2 AHP判断矩阵构建与权重计算

步骤1：构建判断矩阵

采用Saaty 1-9标度法，通过专家评分对10个因素进行两两比较，构建判断矩阵 $A = (a_{ij})_{n \times n}$，满足：

  • $a_{ij} > 0$（正互反性）
  • $a_{ij} = 1/a_{ji}$（互反性）
  • $a_{ii} = 1$（自反性）

步骤2：计算权重向量（几何平均法）

计算判断矩阵每行元素的几何平均值：

$$\bar{w}_i = \sqrt[n]{\prod_{j=1}^{n} a_{ij}}$$

归一化得到权重：

$$w_i = \frac{\bar{w}_i}{\sum_{k=1}^{n} \bar{w}_k}$$

步骤3：一致性检验

最大特征值：

$$\lambda_{\max} = \sum_{i=1}^{n} \frac{[AW]_i}{n \cdot w_i}$$

一致性指标：

$$CI = \frac{\lambda_{\max} - n}{n - 1}$$

一致性比率：

$$CR = \frac{CI}{RI}$$

He et al. (2023) 的计算结果：$\lambda_{\max} = 12.18$，$n = 11$（含地质灾害等额外因素），$CI = 0.118$，$RI = 1.52$，$CR = 0.077 < 0.1$，通过一致性检验。

改进分水岭滤波算法：解决规则方法产生的空间噪声问题。

  1. 从II类（受限通行）开始，找到图像的分水岭
  2. 使用alpha-shape算法计算像素点是否在分水岭内
  3. 如果分水岭内的像素属性为I类（好通行），则滤波为II类
  4. 循环直到IV类（困难通行）

滤波策略遵循单向传播原则：通行能力差的像素可以滤波通行能力好的像素（II→I, III→I+II, IV→I+II+III），反之不行。

3.11.3 RCI修正方程完整推导（2022）

原始Knight方程：

$$\text{RCI} = \exp[a' - b' \ln(\text{MC})]$$

展开为幂律形式：$\text{RCI} = e^{a'} \cdot \text{MC}^{-b'}$

物理含义：土壤强度（RCI）随含水量（MC）的增加而幂律衰减。当 $b'$ 为负值时（如Sand: $b' = -0.815$），$\text{MC}^{-b'} = \text{MC}^{0.815}$，RCI随MC增加而增加——对应砂土含水量增加时毛细力增强的效应。

修正方程推导过程：

步骤1：计算残差RCI（RRCI）——原始Knight方程计算值与实测值之差：

$$\text{RRCI} = \text{RCI}_{\text{computed}} - \text{RCI}_{\text{observed}}$$

步骤2：将RRCI对 $\ln(\text{MC})$ 进行线性回归：

$$\text{RRCI} = a \ln(\text{MC}) + b$$

步骤3：修正后的RCI在原始指数项上叠加线性修正项：

$$\text{RCI}_{\text{modified}} = a\ln(\text{MC}) + b + \exp[a' - b'\ln(\text{MC})]$$

3.11.4 VCI通行性判定准则

车辆锥形指数（VCI）是车辆特定的最低土壤强度要求，与车辆的机动性指数（MI）相关：

$$\text{VCI}_1 = 15.95 + 1.86 \cdot \text{MI} \quad (\text{细粒土，单次通过})$$

$$\text{VCI}_{50} = 26.16 + 2.38 \cdot \text{MI} \quad (\text{细粒土，50次通过})$$

通行性判定准则：

$$\text{RCI} > \text{VCI} \implies \text{可通行}$$

$$\text{RCI} \leq \text{VCI} \implies \text{不可通行}$$

3.11.5 修正方程的物理一致性问题

需要注意修正方程存在以下物理一致性问题：

  • 外推风险：当MC趋近0时，$\ln(\text{MC}) \to -\infty$，对于 $a < 0$ 的土壤类型（如Clay loam），RCI将趋向正无穷；对于 $a > 0$ 的土壤类型（如Sand），RCI将趋向负无穷，均违反物理预期
  • 区域迁移性差：Jaisalmer地区 $R^2 = 0.7754$，但Abohar地区仅 $R^2 = 0.4867$，说明修正系数不具有跨区域迁移性

3.12 刘华军相对特征理论与模糊规则整合

刘华军 (2006) 提出了地形"相对不变性"概念，利用局部邻域内的相对高程提取不受车体颠簸影响的稳定特征。

3.12.1 地形相对不变性的理论证明

基于车体旋转几何关系推导。设车体绕后轮旋转角度 $\gamma$，环境中某点 $P$ 的高程变化为：

$$P'_H = \frac{(P_L + L)\sin\gamma + P_H}{\cos\gamma}$$

其中 $L$ 为车体重心到后轮的距离。

关键推论：当 $\gamma$ 较小时，$P'_L \approx P_L$（距离不变），因此局部邻域内任意两点的相对高程保持不变。这一"相对不变性"为后续基于相对特征的地形分析奠定了理论基础。

3.12.2 起伏度提取——局部方差法

以 $(x_i, y_j)$ 为中心、$\Delta d$ 为半径的邻域 $\Phi_{i,j}$ 内：

$$\bar{f}_{\Phi_{i,j}} = \frac{\sum_{(x_m, y_n) \in \Phi_{i,j}} f(x_m, y_n)}{N_{\Phi_{i,j}}}$$

$$\sigma^2_{\Phi_{i,j}} = \frac{\sum_{(x_m, y_n) \in \Phi_{i,j}} \left[ f(x_m, y_n) - \bar{f}_{\Phi_{i,j}} \right]^2}{N_{\Phi_{i,j}} - 1}$$

其中相对高程 $f(x_m, y_n) - \bar{f}_{\Phi_{i,j}}$ 具有相对不变性。

3.12.3 坡度提取——3×3曲面拟合卷积模板

$$S_X = \frac{1}{8\Delta d} \left[ f(i\!-\!1,j\!-\!1) + 2f(i\!-\!1,j) + f(i\!-\!1,j\!+\!1) - f(i\!+\!1,j\!-\!1) - 2f(i\!+\!1,j) - f(i\!+\!1,j\!+\!1) \right]$$

$$S_Y = \frac{1}{8\Delta d} \left[ f(i\!-\!1,j\!-\!1) + 2f(i,j\!-\!1) + f(i\!+\!1,j\!-\!1) - f(i\!-\!1,j\!+\!1) - 2f(i,j\!+\!1) - f(i\!+\!1,j\!+\!1) \right]$$

$$S = \arctan\sqrt{S_X^2 + S_Y^2}$$

3.12.4 粗糙度提取——分形布朗运动模型

$$E\left[ \left| f(x+\Delta x, y+\Delta y) - f(x,y) \right| \right] = C \cdot \Delta d^{H}$$

取对数后线性回归估计Hurst指数 $H$：

$$\log E\left[ \left| \Delta f \right| \right] = H \cdot \log \Delta d + \log C$$

分形维数 $D_f = n + 1 - H$（$n$ 为空间维数），$D_f$ 越大地形越粗糙。

3.12.5 模糊规则整合

定义三个模糊隶属度函数（马蹄形函数）：

  • 坡度：$\{\text{flat}, \text{sloped}, \text{steep}\}$
  • 起伏度：$\{\text{even}, \text{uneven}\}$
  • 粗糙度：$\{\text{smooth}, \text{rough}, \text{bumpy}\}$

输出可通行性：$\{\text{low}, \text{median}, \text{high}\}$，通过规则表整合。规则表中存在部分"——"（空项），说明规则设计并不完整，这是该方法的一个局限。

3.13.1 PCA方法的完整计算流程

步骤1：数据标准化

$$x'_i = \frac{X_i - \bar{X}_i}{S_i}$$

步骤2：计算相关系数矩阵

$$r_{ij} = \frac{\sum_{k=1}^{n}(x_{ki} - \bar{x}_i)(x_{kj} - \bar{x}_j)}{\sqrt{\sum_{k=1}^{n}(x_{ki}-\bar{x}_i)^2 \cdot \sum_{k=1}^{n}(x_{kj}-\bar{x}_j)^2}}$$

3.14.2 基于Green-Ampt模型的PI通行度指标

累积入渗量（坡度修正）：

$$F(t) = K_s t + \frac{S_f}{1 + \frac{K_s t}{S_f I^* \sin\beta}} \cdot \frac{I^* \sin\beta}{K_s t}$$

3.14.4 风险场建模

采用二维高斯混合概率分布建模风险：

$$p(x,y) = \sum_{k=1}^{K} \frac{TH_k}{2\pi|\Sigma_k|^{1/2}} \exp\left[-\frac{1}{2}(X-\mu_k)^T \Sigma_k^{-1}(X-\mu_k)\right]$$

其中 $TH_k$ 为第 $k$ 个风险点的威胁等级，$\Sigma_k$ 为协方差矩阵，$\mu_k$ 为风险点位置。

其中 $R(x,y)$ 为路径点的风险值，$\omega$ 为风险敏感系数。当风险值高时，平滑权重自动降低，有效防止平滑后路径进入高风险区域。

4. 通行能力图在越野路径规划中的应用方法

4.1 引言

通行能力图（trafficability map）作为越野环境数字化表达的核心载体，其最终价值体现在为路径规划提供科学决策依据。本章系统阐述通行能力图在越野路径规划中的应用方法，涵盖代价图构建、经典算法适配、智能优化算法融合、动态重规划机制、多目标优化框架、结果验证评估及典型应用场景，同时批判性审视当前研究存在的系统性局限。

4.2 通行能力图/代价图的构建方法

通行能力图向代价图（cost surface）的转化是连接环境感知与路径规划的关键桥梁。研究表明，代价图构建需解决栅格化方法、分辨率选择和代价函数设计三个核心问题。

4.2.1 栅格化方法与分辨率选择

主流研究采用规则栅格将连续地理空间离散化。根据最新综述，栅格分辨率通常在1-10米范围内，需平衡计算效率与地形细节表达精度（The Prediction Method and Application of Off-Road Mobility for Ground Vehicles: A Review）。高分辨率（1-2米）适用于精细路径规划，但计算开销大；低分辨率（5-10米）适合大范围快速规划，但可能丢失关键地形特征。

DEM数据从低分辨率向高分辨率插值时，可采用克里金插值法：

$$Z^*(x_0) = \sum_{i=1}^{n} \lambda_i Z(x_i)$$

其中 $\lambda_i$ 为克里金权重，$Z(x_i)$ 为已知点的高程值。

4.2.2 代价函数设计

传统方法将通行能力值直接映射为代价，通行能力越高、代价越低。最新研究提出了更精细的映射策略：一种是线性映射，将归一化通行能力值取反作为代价；另一种是非线性映射，采用指数或对数函数增强对不可通行区域的惩罚（基于复杂地表环境与车辆受力分析的越野机动路径规划方法）。此外，有学者提出风险感知代价图，利用IMU数据和风险价值（VaR）概念生成反映真实行驶风险的连续代价标签，实现可通行/不可通行的二分类向连续风险评估的转变（Learning-based Traversability Costmap for Autonomous Off-road Navigation）。

4.3 经典路径规划算法在越野场景的适配

经典路径规划算法在越野场景中需要结合通行能力图进行针对性改进，主要体现在代价函数融入方式和地形感知扩展两个层面。

4.3.1 A*算法及其启发式函数

A*算法基本原理：A*算法是一种启发式搜索算法，用于在图形平面上从起始节点到目标节点寻找最短路径。其核心是启发式评估函数。

代价函数：

$$f(n) = g(n) + h(n) \tag{1}$$

其中：

  • $f(n)$：节点 $n$ 的总估计代价
  • $g(n)$：从起点到当前节点 $n$ 的实际代价
  • $h(n)$：从节点 $n$ 到目标节点的启发式估计代价（必须满足可采纳性条件：$h(n) \leq h^*(n)$，即估计值不超过实际值）

启发式函数设计：启发式函数的选择直接影响A*算法的搜索效率和路径质量。根据移动方式的不同，常用启发式函数包括以下四种：

（1）曼哈顿距离（Manhattan Distance）——适用于四方向移动的网格环境：

$$h(n) = D \times (|x_N - x_B| + |y_N - y_B|) \tag{2}$$

其中 $D$ 为相邻节点间的移动代价，$(x_N, y_N)$ 为当前节点坐标，$(x_B, y_B)$ 为目标节点坐标。

（2）欧几里得距离（Euclidean Distance）——适用于任意方向移动的环境：

$$h(n) = D \times \sqrt{(x_N - x_B)^2 + (y_N - y_B)^2} \tag{3}$$

（3）切比雪夫距离（Chebyshev Distance）——适用于八方向移动的网格环境：

$$h(n) = D \times \max(|x_N - x_B|, |y_N - y_B|) \tag{4}$$

（4）对角线距离（Diagonal Distance）——适用于八方向移动，考虑对角线代价：

$$h(n_1) = \min(|x_N - x_B|, |y_N - y_B|) \tag{5}$$ $$h(n_2) = |x_N - x_B| + |y_N - y_B| \tag{6}$$ $$h(n) = D_2 \times (\sqrt{2} \times h(n_1) + D \times (h(n_2) - 2 \times h(n_1))) \tag{7}$$

其中 $D_2$ 为对角线移动代价。

改进的启发式函数：针对不同方向移动代价不同的情况，改进的启发式函数为：

当 $|x_N - x_B| \geq |y_N - y_B|$ 时：

$$h(n) = D_1 \times |y_N - y_B| + D_2 \times |x_N - x_B| \tag{8}$$

当 $|x_N - x_B| < |y_N - y_B|$ 时：

$$h(n) = D_2 \times |y_N - y_B| + D_1 \times |x_N - x_B| \tag{9}$$

其中最优参数值为 $D_1 = 10$，$D_2 = 8$。

4.3.2 通行能力导向的A*改进

传统A*算法以欧氏距离为代价函数，规划路径可能穿越不可通行区域。最新研究提出通行能力导向的A*算法，将代价函数改进为 $f(n) = \Sigma p_i - (p_{min} \times l_n)$，其中 $\Sigma p_i$ 为路径累积通行能力，$p_{min}$ 为剩余路径最小通行能力，$l_n$ 为欧氏距离。该改进使算法优先选择通行能力高且通往目标路径条件较好的节点，在复杂地形中优势明显（Construction of dynamic trafficability map for unmanned vehicles considering multiple environmental factors and path planning）。此外，有研究将A*与动态窗口法（DWA）结合，实现全局规划与局部避障的协同，在应急救援场景中取得良好效果（Integrated A* and DWA algorithms for emergency rescue path planning）。

4.3.3 基于AHP权重模型的通行时间-成本最优A*算法

孙鹏飞提出的通行时间-成本最优A*算法采用层次分析法（AHP）确定各通行影响因素的权重，将地形因素综合融入A*代价函数。

AHP一致性指标：

$$CI = \frac{\lambda - n}{n - 1} \tag{10}$$

速度系数计算：车辆在越野环境中的速度系数由四个因素加权计算：

其中 $W_G$、$W_S$、$W_Q$、$W_T$ 分别为高程、坡度、起伏度、地表覆盖的权重，$w_{Gi}$、$w_{Si}$、$w_{Qi}$、$w_{Ti}$ 分别为各因素在特定区间的归一化值。

4.3.4 基于DEM的改进A*越野路径规划

在越野路径规划中，传统A*算法仅考虑距离因素，改进后需综合考虑地形因素。通行成本函数通常包含四项：

  • 坡度成本：地形坡度直接影响车辆通行难度
  • 高程成本：海拔变化影响发动机功率
  • 地表覆盖成本：不同地表类型（耕地、林地、草地、裸地、人工地表）影响通行速度
  • 粗糙度成本：地表微观起伏影响行驶平稳性

改进的代价函数：

$$F(n) = G(n) + H(n) \tag{17}$$

其中 $G(n)$ 为从起点到节点 $n$ 的累积通行成本（综合考虑坡度、高程、地表覆盖和粗糙度），$H(n)$ 为启发式估计。

4.3.5 A*算法最优性理论与Weighted A*失效条件

A*最优性理论：A*算法在满足以下条件时保证找到最优路径：

  • 可采纳性（Admissibility）：启发式函数 $h(n)$ 永远不高估到达目标的实际代价，即 $h(n) \leq h^*(n)$ 对所有节点 $n$ 成立
  • 一致性（Consistency）：对于任意节点 $n$ 和其后继节点 $n'$，满足三角不等式 $h(n) \leq c(n, n') + h(n')$，其中 $c(n, n')$ 为从 $n$ 到 $n'$ 的实际代价

一致性条件比可采纳性更强——一致的启发式函数一定是可采纳的。在一致性条件下，A*算法首次到达目标节点时即找到最优路径，无需重新打开已扩展节点。

Weighted A*及其失效条件：Weighted A*通过引入权重因子 $\epsilon > 1$ 对启发式函数加权：

$$f(n) = g(n) + \epsilon \cdot h(n)$$

这可以加速搜索，但牺牲了最优性保证。Weighted A*保证找到的路径代价不超过最优代价的 $\epsilon$ 倍（$\epsilon$-次优性）。然而，Weighted A*在以下条件下可能失效或严重退化：

  • 当 $\epsilon$ 过大时，算法过于贪心，可能在局部区域大量扩展节点，反而降低效率
  • 当启发式函数在越野环境中严重偏离实际代价时（例如，未考虑通行能力差异），加权放大了估计误差
  • 当地形存在狭窄可行通道时，Weighted A*可能因贪心倾向而跳过最优可行路径

4.3.6 Dijkstra算法

Dijkstra算法作为A*的特例（即 $h(n)=0$），在越野环境中通常作为基准方法。其优势在于保证找到全局最优路径，但计算效率较低，适用于小范围精确规划。

4.3.7 RRT与RRT*算法

RRT（快速扩展随机树）算法在高维状态空间和复杂运动约束下表现出色。研究表明，RRT能有效处理车辆动力学约束和非完整性约束，适合复杂越野地形的探索性规划（The Prediction Method and Application of Off-Road Mobility for Ground Vehicles: A Review）。然而，RRT规划路径通常不最优，需要后处理优化。

启发式采样策略：传统RRT算法采用均匀随机采样，效率较低。改进的启发式采样策略通过语义分割结果引导采样方向：

$$X_r = \begin{cases} X_h & r < \rho \\ X_u & r \geq \rho \end{cases} \tag{18}$$

其中 $X_r$ 为采样点，$X_h$ 为启发式采样点（基于语义分割结果的可行区域），$X_u$ 为均匀随机采样点，$r$ 为 $[0,1]$ 之间的随机数，$\rho = 0.7$ 为启发式采样概率阈值。当 $r < \rho$ 时采用启发式采样，否则采用均匀随机采样，保证算法的探索能力。

RRT*算法改进：RRT*算法在RRT基础上增加了路径代价优化，引入重布线（rewiring）机制，当找到更优路径时更新父节点，保证渐近最优性（asymptotic optimality）。

4.3.8 D* Lite算法

D* Lite算法专为动态环境设计，能在环境变化时高效重规划路径。最新研究将其与通行能力图结合，当土壤湿度、气象条件等导致通行能力变化时，仅更新受影响区域的代价并局部调整路径，显著降低计算开销（A Comprehensive Study of Recent Path-Planning Techniques in Dynamic Environments）。

4.4 智能优化算法在越野路径规划中的应用

元启发式智能优化算法在越野路径规划中展现出独特优势，尤其适用于多约束、非线性的复杂优化问题。

4.4.1 遗传与蚁群混合路径规划算法（GA-ACO）

GA-ACO混合算法将遗传算法（GA）的快速收敛能力与蚁群算法（ACO）的正反馈机制相结合（武小年等，《计算机应用研究》2020年）。算法分为两个阶段：首先利用遗传算法快速生成优化解，然后将遗传算法的优化解转换为蚁群算法的初始信息素，通过蚁群算法进一步优化。

选择因子设计：为提高初始种群生成效率，构造选择因子 $G_{ij}$，使节点选择更倾向于终点方向：

$$G_{ij} = \frac{[2 + (x_j - x_i) + (y_j - y_i)]}{2m} \tag{19}$$

其中 $q_i$ 为当前节点，$U_i$ 为 $q_i$ 的可选节点集，$m$ 为 $U_i$ 中的节点数量，$q_j \in U_i$，$G_{ij}$ 表示从 $q_i$ 通往终点时途经 $q_j$ 的选择因子。该选择因子具有重要性质：当节点越靠近终点时，其选择因子的值越大，被选择的概率越大。

适应度函数：以路径长度的倒数作为适应度指标：

$$H(i, i-1) = |z_i - z_{i-1}| \tag{20a}$$ $$D(i, i-1) = \sqrt{H(i, i-1)^2 + L(i, i-1)^2} \tag{20b}$$ $$LEN(l) = \sum_{i=1}^{n} D(i, i-1) \tag{20c}$$ $$F(l) = \frac{1}{LEN(l)} \tag{21}$$

其中 $L(i, i-1)$ 为相邻节点的平面直线距离，$H(i, i-1)$ 为相邻节点的高程差，$D(i, i-1)$ 为相邻节点的空间直线距离，$LEN(l)$ 为路径总长度。

自适应信息素初始化：

$$\tau_{(i)g} = \sum_{l=1}^{s} \frac{Q_g}{LEN(l)} \tag{22a}$$ $$\tau_{(i)s} = \tau_c + \tau_{(i)g} \tag{22b}$$ $$Q_g = \frac{1}{z} \times \sum_{v=1}^{z} LEN(v) \tag{22c}$$

其中 $z$ 为通过遗传过程得到的优化解总数，$v$ 为某个优化解，$s$ 为经过节点 $q_i$ 的路径总数，$l$ 为经过节点 $q_i$ 的一条路径，$Q_g$ 为 $\tau_{(i)g}$ 的自适应参数。

ACS策略——蚂蚁寻路中下一节点选择规则：

$$T = \arg\max\{[\tau_{ij}(t)] \times [\eta_{ij}]\} \quad q < p \tag{23a}$$ $$P_{ij}(t) = \frac{[\tau_{ij}(t)]^\alpha \times [\eta_{ij}]^\beta}{\sum_{q_j \in U_i} [\tau_{ij}(t)]^\alpha \times [\eta_{ij}]^\beta} \quad \text{otherwise} \tag{23b}$$

其中 $\tau_{ij}(t)$ 为 $t$ 时刻节点 $q_i$ 到 $q_j$ 的信息素浓度，$\alpha$ 为信息素权重因子，$\eta_{ij}$ 为启发式因子（节点 $q_i$ 到 $q_j$ 空间直线距离的倒数），$\beta$ 为启发式因子权重，$q$ 为随机数 $q \in (0,1)$，$p$ 为阈值 $p \in (0,1)$。

信息素自适应更新策略：

$$D = \frac{1}{a} \times \sum_{k=1}^{a} LEN(k) \tag{24a}$$ $$Q = \lambda \times D \tag{24b}$$ $$\Delta\tau_{kij} = \frac{Q}{LEN(k)} \tag{24c}$$ $$\tau_{ij}(t+1) = (1-\rho) \times \tau_{ij}(t) + \rho \times \sum_{k=1}^{c} \Delta\tau_{kij}, \quad \rho \in (0,1) \tag{25}$$

其中 $\rho$ 为信息素的蒸发系数，$c$ 为经过节点 $q_i$ 到节点 $q_j$ 的蚂蚁总数，$\lambda$ 为更新系数，$a$ 为本次迭代中蚂蚁搜索的路径总数。

4.4.3 蚁群算法（ACO）与其他智能优化

蚁群算法（ACO）通过模拟蚂蚁觅食行为实现路径优化。最新研究提出改进蚁群算法，通过优化信息素更新规则、引入精英策略和自适应参数调整，提升收敛速度和路径质量。实验表明，改进算法在越野环境中的路径规划成功率提升15-20%，计算时间减少30%（Path planning for autonomous vehicles based on the improved ant colony）。

遗传算法（GA）通过选择、交叉、变异操作实现路径优化。研究表明，将通行能力图作为适应度函数的一部分，能使遗传算法优先进化出高通行能力的路径。最新研究提出动态遗传蚁群算法（DGA-ACO），根据环境复杂度动态调整算法参数，在简单地形中快速收敛，在复杂地形中充分探索（DGA-ACO: Enhanced Dynamic Genetic Algorithm—Ant Colony）。

粒子群优化（PSO）算法通过模拟鸟群觅食行为实现路径优化。其优势在于参数少、收敛快，但易陷入局部最优。有研究将PSO与模拟退火算法结合，利用模拟退火的概率突跳特性帮助PSO跳出局部最优，在越野路径规划中取得较好效果。

4.4.4 DWA动态窗口路径规划

DWA（Dynamic Window Approach）是一种局部路径规划算法，其原理为：在速度空间中采样多组 $(v, \omega)$ 速度对，根据运动模型模拟这些速度下的轨迹，通过评价函数选择最优速度对。

评价函数：

$$G(v, \omega) = \alpha \cdot \text{heading}(v, \omega) + \beta \cdot \text{dist}(v, \omega) + \gamma \cdot \text{vel}(v, \omega) \tag{34}$$

其中 $\text{heading}$ 为朝向目标的程度，$\text{dist}$ 为与最近障碍物的距离，$\text{vel}$ 为前进速度，$\alpha$、$\beta$、$\gamma$ 为权重系数。

4.5 动态路径规划与实时重规划

越野环境具有显著的时空动态性，气象条件、土壤湿度、植被状态等因素随时间变化，要求路径规划系统具备动态更新和实时重规划能力。

4.5.1 动态通行能力图更新机制

最新研究将环境因素分为短期不变因素（高程、坡度、地貌等）和动态变化因素（土壤、降雨、降雪、风力等），建立时变通行能力模型。当动态因素变化超过阈值时，系统自动更新受影响区域的通行能力值，并触发路径重规划（Construction of dynamic trafficability map for unmanned vehicles considering multiple environmental factors and path planning）。

4.5.2 实时重规划策略

实时重规划策略主要有三种：一是全局重规划，当环境发生重大变化时重新规划整个路径，计算开销大但保证全局最优；二是局部重规划，仅调整受影响路段，计算效率高但可能牺牲全局最优性；三是分层重规划，先进行粗粒度全局规划，再进行细粒度局部调整，平衡全局性与实时性（Multi-Sample Long Range Path Planning under Sensing）。研究表明，分层重规划在越野环境中综合性能最佳。

4.5.3 传感器融合与环境感知

通过融合激光雷达、摄像头、IMU等多种传感器数据，实时感知地形变化和障碍物出现，为路径重规划提供最新环境信息。最新研究提出协调路径规划方法，通过多传感器融合实现环境状态的实时估计，在越野自主车辆中取得良好效果（Coordinated path planning for autonomous ground vehicles in off-road）。

4.6 多目标路径规划

越野路径规划通常需要同时考虑多个相互冲突的目标，如路径最短、通行能力最高、能耗最低、安全性最好、时间最短等，形成多目标优化问题。

4.6.1 基于Pareto最优的多目标路径规划框架

最新研究提出基于Pareto最优的多目标路径规划框架。该框架将多个目标同时纳入优化过程，生成一组Pareto最优解集，决策者可根据实际需求从中选择最合适的路径。研究表明，在越野环境中，Pareto最优解集通常包含5-15条具有不同权衡特性的路径（Multi-Objective Off-Road Path Planning on the Basis of the Pareto Optimality）。

NAMOA*（New Anytime Multi-Objective A*）算法是多目标路径规划中的代表性方法，其核心思想是将A*算法扩展至多目标场景：

对于 $k$ 个目标，每个节点 $n$ 维护一组非支配代价值向量：

$$\mathbf{g}(n) = \{g_1(n), g_2(n), \ldots, g_k(n)\}$$

其中 $g_i(n)$ 表示从起点到节点 $n$ 在第 $i$ 个目标上的累积代价。启发式函数也需要扩展为向量形式：

$$\mathbf{h}(n) = \{h_1(n), h_2(n), \ldots, h_k(n)\}$$

节点优先级由标量化函数确定，例如加权Tchebycheff函数：

$$f(n) = \max_{i=1}^{k} \{w_i \cdot |g_i(n) + h_i(n) - z_i^*|\}$$

其中 $w_i$ 为第 $i$ 个目标的权重，$z_i^*$ 为第 $i$ 个目标的理想点值。NAMOA*算法保证找到Pareto最优解集中的所有非支配路径。

4.6.2 目标权重确定方法

目标权重确定方法主要有三种：一是主观赋权法，如层次分析法（AHP），依赖专家经验；二是客观赋权法，如熵权法，基于数据本身特性；三是组合赋权法，综合主观与客观方法。最新研究提出自适应权重调整方法，根据环境特征和车辆状态动态调整各目标权重，提升路径规划的适应性（A Multi-Objective Path Planning Framework in Off-Road）。

4.6.3 考虑能耗的路径规划

考虑能耗的路径规划是多目标优化的重要方向。研究表明，在越野环境中，路径距离最短并不一定能耗最低，因为坡度、地表粗糙度、土壤类型等因素显著影响车辆能耗。最新研究提出考虑能耗的履带车辆路径规划方法，将能耗模型集成到路径规划过程中，在保证通行性的前提下优化能耗（考虑能耗的无人驾驶履带车辆全局路径快速规划方法）。

4.7 路径规划结果的验证与评估

路径规划结果的合理性验证是确保规划方法有效性的关键环节，主要包括仿真验证、实地试验和与已有路网对比三种方法。

仿真验证方面，利用高精度越野环境仿真平台（如Gazebo、CARLA等）对规划路径进行虚拟测试。最新研究在Gazebo仿真中对改进A*算法进行验证，结果表明规划路径的总通行能力和平均通行能力均优于传统算法，且能有效避开不可通行区域（Construction of dynamic trafficability map for unmanned vehicles considering multiple environmental factors and path planning）。

实地试验是最直接的验证方法。通过实际车辆在越野环境中行驶，记录路径跟踪精度、通行时间、能耗等指标，评估规划路径的可行性。最新研究在复杂越野地形中进行实地试验，结果表明改进算法规划的路径在实际行驶中表现出更高的安全性和效率（基于复杂地表环境与车辆受力分析的越野机动路径规划方法）。

与已有路网对比通过将规划路径与实际存在的道路网络进行对比，评估路径的合理性。研究表明，规划路径应尽量与已有路网在关键路段保持一致，但在无路区域应能自主规划出可行路径（基于地形数据的长距离越野路径快速规划方法研究）。

4.8 典型应用案例分析

通行能力图与路径规划技术在不同应用场景中展现出重要价值。

军事机动方面，越野路径规划对部队快速部署、后勤补给和战术机动至关重要。北约下一代参考机动性模型（NG-NRMM）为军事越野机动提供了标准化评估框架，基于NG-NRMM的路径规划系统已在多个国家军队中应用（Next-Generation NATO Reference Mobility Model (NG-NRMM)）。

应急救援方面，在地震、洪水、泥石流等自然灾害发生后，道路网络可能中断，越野路径规划成为救援力量到达灾区的关键。最新研究提出应急救援路径规划方法，综合考虑通行能力、时间紧迫性和救援资源分布，为救援车辆规划最优路径（Integrated A* and DWA algorithms for emergency rescue path planning）。

自动驾驶方面，越野自动驾驶车辆需要自主感知环境、规划路径并控制行驶。最新研究提出考虑驾驶舒适性和距离的路径规划方法，在保证安全性的前提下优化乘坐舒适性，为越野自动驾驶提供更人性化的路径选择（Path Planning Method for Autonomous Vehicles on Off-Road Terrain Considering Driving Comfort and Distance）。

4.9 当前路径规划应用中的核心问题

尽管路径规划技术取得显著进展，但在实际应用中仍面临多项挑战。

计算效率与实时性瓶颈是首要挑战。高分辨率通行能力图和复杂优化算法导致计算开销大，难以满足实时路径规划需求。最新研究提出多尺度加速方法，通过局部到全局的多尺度处理，在保证规划质量的前提下提升计算效率（LGMM: Local-to-global multi-scale accelerated mobility map prediction for unmanned ground vehicles）。

尺度适应性问题指路径规划方法在不同空间尺度下的性能差异。小范围精细规划与大范围宏观规划对算法的要求不同，现有方法难以同时满足多尺度需求。研究表明，分层规划框架能有效缓解这一问题，但增加了系统复杂性。

车辆-环境耦合不足指现有方法对车辆动力学特性与地形交互的考虑不够充分。最新研究提出车辆受力分析方法，将车辆动力学模型集成到路径规划过程中，提升规划路径的可执行性（基于复杂地表环境与车辆受力分析的越野机动路径规划方法）。

动态环境适应性不足指现有方法对环境时变特性的响应能力有限。虽然动态重规划技术有所发展，但在快速变化的环境中仍显不足，需要更高效的环境感知和路径更新机制。

4.10 批判性思考与改进方向

路径规划与通行能力建模的脱节问题是当前研究的系统性局限。通行能力建模侧重于环境静态或准静态特性的评估，而路径规划需要考虑动态交互和实时决策。两者在时间尺度、空间分辨率和优化目标上存在不匹配，导致规划路径在实际执行中可能偏离预期。

现有方法的系统性局限包括：一是环境模型简化，现有方法通常假设环境因素相互独立，忽略了因素间的复杂交互；二是车辆模型简化，对车辆动力学特性的建模不够精细，影响规划路径的可执行性；三是验证方法局限，仿真验证与实际环境存在差距，实地试验成本高、风险大。

未来改进方向包括：一是深度学习与强化学习的融合，利用深度学习处理高维环境信息，强化学习实现端到端的路径规划；二是数字孪生技术的应用，构建高保真虚拟环境，实现规划-验证-优化的闭环；三是多智能体协同规划，考虑多车辆协同作业时的路径协调与冲突消解；四是不确定性建模与鲁棒规划，显式考虑环境感知和车辆控制的不确定性，提升规划路径的鲁棒性。

小结

通行能力图在越野路径规划中的应用已形成从代价图构建、算法适配、动态重规划到多目标优化、结果验证的完整技术体系。经典算法通过代价函数改进和地形感知扩展适应越野场景，智能优化算法为复杂约束问题提供新思路，动态重规划机制提升环境适应性，多目标优化框架满足实际应用的多维权衡需求。然而，计算效率、尺度适应性、车辆-环境耦合和动态适应性仍是主要挑战。未来研究应着重解决路径规划与通行能力建模的脱节问题，推动深度学习、数字孪生和多智能体协同等新技术的应用，提升越野路径规划系统的智能化水平和实际应用价值。

5. 现有研究不足与前沿技术趋势

5.1 引言

前四章系统梳理了越野通行能力建模的理论基础（第一章）、单要素提取与量化方法（第二章）、多因子综合评估模型（第三章）及路径规划应用（第四章）。本章旨在基于前述分析，系统审视当前研究存在的不足与挑战，并结合最新技术进展展望前沿趋势，为后续研究提供方向性参考。

5.2 现有研究的系统性不足

5.2.1 权重确定的主观性与客观性矛盾

多因子综合评估模型的核心在于各影响因子权重的确定，但这一过程长期面临主观性与客观性的根本矛盾。事实上，层次分析法（AHP）虽广泛应用于权重确定，但其依赖专家判断的本质使其一致性比率CR<0.1的阈值标准本身具有一定主观性 (Construction of dynamic trafficability map)。参考比较经验阈值法（RCTM）等客观方法可部分弥补这一不足，但RCTM同样依赖预先设定的经验阈值区间 (RCI重塑锥指数通行性评估)。有学者认为，主观赋权反映领域知识但缺乏可复现性，客观赋权保证数据驱动但可能忽视物理意义，两者在理论层面尚未找到完美的平衡点。MCDA多准则决策方法虽提供了更为系统的评估框架，但权重敏感性分析表明，即使权重发生10%的偏移，评估结果也可能出现显著差异 (Off-road trafficability using MCDA)。

5.2.2 静态模型与时变环境的矛盾

传统通行能力建模基于静态地理信息数据，假设环境要素在评估周期内保持不变。然而，越野环境具有显著的时空动态性：土壤含水量随降雨变化，植被覆盖随季节更替，积雪和冻土状态受气象条件直接影响。研究表明，考虑气象因素的时空变化可以构建动态通行能力图，但当前大多数研究仍停留在准静态阶段 (Construction of dynamic trafficability map)。动态模型虽已出现，但其对实时气象数据的依赖和高频率更新带来的计算负担，限制了实际部署 (越野移动性预测方法综述)。

5.2.3 车辆类型差异化不足

不同车辆类型——轮式、履带式、多轴车辆——在几何通过性、力学通过性和越野速度方面存在显著差异。然而，许多研究采用统一的车辆参数模型，未充分考虑车辆特异性。NG-NRMM通过基于物理的建模方法部分解决了这一问题，但其对详细车辆参数（悬挂特性、轮胎规格、质心位置等）的需求限制了通用性 (NG-NRMM标准)。越野过程与行驶装置设计的最新综述指出，车辆-地面交互是一个高度耦合的非线性系统，当前简化假设难以准确反映真实车辆行为 (越野过程与行驶装置设计科学问题)。

5.2.4 数据精度与模型精度的脱节

高精度数据（如1-2米分辨率DEM）能更好地捕捉地形细节，但导致计算开销呈指数增长。数据显示，当栅格分辨率从10米提高到1米时，计算量可增加100倍以上。LGMM多尺度机动地图预测方法通过局部到全局的多尺度处理在一定程度上缓解了这一矛盾，但多尺度信息损失问题仍然存在 (LGMM多尺度机动地图预测)。

5.2.5 缺乏统一基准与标准化评估

不同研究采用不同的数据集、评估指标和实验设置，导致方法之间的横向比较极为困难。有学者指出，当前研究缺乏统一的基准测试平台和标准化评价体系 (越野移动性预测方法综述)。Verti-Bench越野通行性基准测试集的出现为这一问题提供了解决方案，但其应用范围仍有限 (Verti-Bench)。

5.2.6 实验验证不足

现有验证方法主要依赖仿真验证和有限的实地试验。然而，仿真环境与真实越野环境之间存在本质差异——仿真难以完全模拟土壤的非线性力学行为、植被的随机干扰和气象的突变效应。数据显示，超过70%的越野通行能力研究停留在仿真验证阶段，缺乏实地验证 (越野移动性预测方法综述)。

5.2.7 路径规划与通行能力建模的脱节

通行能力建模与路径规划作为两个相对独立的研究领域，其集成度仍有待提升。事实上，许多研究专注于通行能力评估算法的改进，而将其在路径规划中的应用视为次要目标；反之，路径规划研究往往将通行能力图视为静态输入，未充分利用其动态更新特性 (基于复杂地表环境与车辆受力分析的越野机动路径规划方法)。

5.3 文献精读发现的深层方法论问题

基于对Suvinen (2006)、He (2023)、Pundir系列 (2020/2021/2022)、刘华军 (2006)、Xu & Wang (2020)、赵得全 (2025)、Zevenbergen (1987)、万晨、黄鲁峰、张文诗等文献的深度精读分析，本节系统归纳各类方法的深层方法论问题。

5.3.1 物理力学模型的局限性（以Suvinen为代表）

数据精度限制：Suvinen使用25m DEM分辨率，对微地形分析不够精细；土壤图1:20000比例尺空间分辨率有限；土壤参数基于北欧条件，跨区域应用需重新标定。

模型简化问题：轮载分配模型过于简化（固定60/40比例），虚拟轮半径模型的物理假设需要验证；加法模型在极端地形可能存在耦合误差——各阻力分量之间可能存在非线性交互，简单的力相加可能不够准确。

验证严重不足：仅有案例研究，缺乏定量验证；没有与实际行驶数据对比，没有与其他方法对比。

5.3.2 AHP-WIC方法的主观性问题（以He 2023为代表）

AHP主观性：判断矩阵依赖专家经验，存在主观偏差；权重结果可能因专家不同而变化；缺乏客观的权重确定方法（如熵权法、CRITIC法等）的系统对比。

信息含量赋值问题：信息含量赋值（1-9）缺乏物理依据；赋值规则是主观的，不同专家可能给出不同赋值；缺乏与实际通行能力的定量关系验证。

分类方法问题：Jenks自然断点法基于数值统计分布，可能忽略物理意义；分类数量（3类或4类）是人为设定的；不同区域的分类阈值差异很大，缺乏统一标准。

计算效率问题：342,144条规则的生成和计算量巨大；多层分水岭滤波计算复杂度高；大规模区域应用可能面临计算瓶颈。

5.3.3 经验修正方程的物理一致性问题（以Pundir为代表）

修正RCI方程的极限行为异常：对于Sand（$a = 12.19$），当MC → 0时，$a\ln(\text{MC}) \to -\infty$，修正方程给出负无穷大的RCI——但极度干燥的砂土应具有很高的承载力。对于Clay loam（$a = -215$），当MC增大时，指数项 $\exp[a' - b'\ln(\text{MC})]$ 迅速增大——但饱和粘土的承载力应极低。

区域迁移性差：Jaisalmer地区 $R^2 = 0.7754$，但Abohar地区仅 $R^2 = 0.4867$，说明修正系数不具有跨区域迁移性，每个新区域都需要大量实测数据重新标定。

累积评分法权重不合理：土壤湿度权重仅5/30 = 16.7%，严重低估了含水量对土壤承载力的影响。根据Terzaghi理论，粘土含水量从干燥到饱和可导致承载力下降90%以上，但评分法中仅损失13.3%的总分。敏感性分析显示，湿度的敏感性指数最低（0.96），远低于土地利用（2.46），与物理直觉不符。

5.3.4 相对不变性到绝对可通行性的跳跃（以刘华军为代表）

特征与通行性映射不明确：论文仅通过模糊规则将特征映射到三级可通行性，但未建立特征值与车辆物理能力之间的定量关系。

车辆特性未纳入：不同车辆在同一地形上的通行能力差异巨大，但方法完全不考虑车辆参数。

特征提取尺度单一：起伏度和粗糙度的计算仅在单一尺度（2.4m圆域）下进行，未讨论多尺度特征提取的效果对比。

5.3.5 PCA降维方法的适用性问题（以Xu 2020为代表）

线性假设的局限：PCA本质上是线性降维方法，当地形因子间存在非线性关系时可能失效。

综合指数Tr的信息损失：虽然Tr解释了87.623%的方差，但丢失的12.377%方差中可能包含对通行性评估至关重要的信息（如极端地形特征）。

缺乏通行性验证：Tr指数仅通过ARCGIS可视化进行定性验证，缺乏与实际车辆通行数据的对比验证，未给出Tr值与通行能力之间的定量关系。

5.3.6 多因子耦合的简化问题（以赵得全为代表）

线性叠加假设：坡度、地物、降雨三个因子的通行权重采用叠加分析，忽略了因子间的非线性交互。实际上，陡坡+暴雨的组合效应远大于两者单独效应之和。

PI指标的物理意义不完整：PI值与车辆通行能力之间的定量关系未建立；未给出PI值到通行度的映射函数。

启发式函数的可采纳性问题：改进后的 $H(node, goal) = D(node, goal) + T(node, goal)$ 中，$T$ 包含了通行时间信息但仅基于直线距离估算，可能高估实际代价，丧失A*算法的最优性保证。

5.3.7 DEM方法的算法局限

Zevenbergen D8算法：只能将水流分配给8个邻域中坡度最陡的一个方向，无法处理分流情况，在平坦区域可能导致汇水面积计算错误。仅通过行式打印机地图进行定性验证，缺乏定量对比。

万晨CNN地貌划分：总体精度79.04%，极高山用户精度仅61.25%，高山仅70.37%。CNN架构较简单（仅2层卷积+2层池化），卷积核21×21较大，可能丢失局部细节。分块处理会破坏地貌完整性。

黄鲁峰方法：各环境因子的影响系数和权重需要大量实验数据支撑，部分参数来源于经验估计。因子间非线性交互作用建模不够深入。

张文诗方法：许多影响系数来自特定条件下的试验，泛化能力有限；隶属度函数设计较为简单；缺乏基于GIS的空间分析和可视化能力。

5.4 数据层面的挑战

遥感数据的时空分辨率限制是基础性挑战。NASADEM等全球DEM数据的空间分辨率通常为30米，对精细车辆动力学分析存在不足；高时间分辨率的土壤含水量遥感产品虽已出现，但空间分辨率往往较低 (无道路环境车辆支承通过性全遥感反演方法)。多源数据融合面临坐标系统不一致、时间基准不匹配、数据质量参差不齐等问题。实地验证数据的稀缺尤为突出——越野环境的恶劣条件和安全风险使得大规模实地数据采集极为困难。

5.5 模型层面的挑战

因素交互作用建模不足是当前模型的结构性缺陷。现有方法通常假设各环境因素独立作用于通行能力，通过加权叠加得到综合结果。然而，现实中因素间存在复杂的非线性交互——例如，降雨既改变土壤含水量又影响地表粗糙度，其交互效应远大于单一因素效应之和。

跨尺度建模困难指从厘米级车辆-地面接触到公里级区域通行能力评估之间的尺度鸿沟，现有方法难以在不同尺度之间进行有效的信息传递和一致性保证 (越野过程与行驶装置设计科学问题)。

不确定性量化缺失尤为关键——现有方法大多输出确定性评估结果，未显式考虑输入数据误差、模型参数不确定性和环境随机性对最终结果的影响。最新研究开始探索神经过程在越野地形建模中的不确定性量化能力 (Uncertainty-aware Elevation Modeling via Neural Processes)。

5.6 应用层面的挑战

从实验室到实际部署的鸿沟是制约技术转化的核心瓶颈。实时性与精度的矛盾贯穿于整个技术链条——高精度模型需要大量计算时间，而实际应用要求毫秒级响应。车辆-环境耦合建模不足指现有方法对车辆动力学特性与地形交互的考虑不够精细 (基于复杂地表环境与车辆受力分析的越野机动路径规划方法)。

5.7 前沿技术趋势

5.6.1 深度学习与多模态融合

深度学习与强化学习正在重塑越野通行能力建模范式。RoadRunner框架展示了从传感器数据直接预测可通行性和高程图的端到端深度学习方法，相比传统方法可将推理延迟降低约4倍 (RoadRunner)。EVORA方法提出了基于证据深度学习的不确定性感知可通行性框架，联合建模内禀不确定性和认知不确定性 (EVORA)。跨模态特征融合网络通过整合遥感影像与地理地质数据进一步提升了评估精度 (跨模态特征融合网络越野通行性评估)。

5.6.2 动态实时通行能力图

动态实时通行能力图是应对环境时变性的关键技术。最新研究将环境因素分为短期不变因素（高程、坡度、地貌等）和动态变化因素（土壤、降雨、降雪、风力等），建立时变通行能力模型。当动态因素变化超过阈值时，系统自动更新受影响区域的通行能力值，并触发路径重规划 (Construction of dynamic trafficability map)。边缘计算与实时化通过将计算任务部署到车载平台或近端边缘服务器，解决云计算的延迟问题。

5.6.3 全遥感反演方法

全遥感反演方法代表了数据获取范式的转变。无道路环境车辆支承通过性全遥感反演方法实现了从遥感数据到车辆参数的完整链条，无需实地采样即可评估土壤承载力，车速预测精度优于90% (全遥感反演方法)。

5.6.4 车辆-环境耦合建模

车辆-环境深度耦合模型将车辆动力学仿真与地理环境模型进行紧耦合集成。最新研究将车辆受力分析与地形数据融合，实现面向特定车辆的定制化通行能力评估 (基于复杂地表环境与车辆受力分析)。

5.6.5 多目标优化与Pareto前沿

多目标优化与Pareto前沿为同时考虑多个相互冲突的目标提供了理论框架。最新研究提出了基于Pareto最优的越野路径规划方法，能够在路径长度、通行能力、能耗、安全性等多个目标之间寻找最优平衡 (Pareto Optimality Off-Road Path Planning)。

5.6.6 大规模并行计算与实时规划

GPU加速技术能够显著提升路径规划算法的计算速度，有研究表明GPU加速可使RRT算法的速度提升10-100倍。分层规划策略通过粗粒度全局规划与细粒度局部调整的结合，平衡全局最优性与实时性需求。

5.6.7 基础模型在地理空间分析中的应用

基础模型（Foundation Models）是最值得关注的新兴方向。数据表明，2021年至2024年间已出现58个遥感视觉基础模型 (Towards responsible geospatial foundation models)。这些模型通过在海量多模态遥感数据上预训练，可迁移至包括通行性评估在内的多种下游任务，有望从根本上改变当前依赖手工特征工程和小样本训练的研究范式。

5.8 未来研究方向建议

五大重点研究方向

  1. 动态实时通行能力图：融合时变气象数据、实时传感器信息和动态更新机制，实现通行能力的连续评估与预测。
  2. 车辆-环境深度耦合模型：将车辆动力学仿真与地理环境模型进行紧耦合集成，输出面向特定车辆的定制化通行能力评估。
  3. 不确定性感知的鲁棒路径规划：显式量化并传播各类不确定性源，使规划结果在最坏情况下仍具备可接受的安全裕度。
  4. 跨领域知识迁移：利用迁移学习和域适应技术，将一个地形环境中训练的模型快速适配至新的地理区域。
  5. 统一基准与标准化评估体系：建立涵盖多种地形类型、车辆类型和评估指标的基准测试平台。

5.9.1 物理约束的RCI修正方程

针对Pundir修正方程在极端条件下行为不合理的问题，文献提出引入物理约束的修正方程：

$$\text{RCI} = \text{RCI}_{\text{dry}} \cdot \exp\left[-\beta \cdot \left(\frac{\text{MC} - \text{MC}_{\text{dry}}}{\text{MC}_{\text{sat}} - \text{MC}_{\text{dry}}}\right)^\gamma\right] + \epsilon$$

其中 $\text{RCI}_{\text{dry}}$ 为干燥状态RCI（可实测），$\text{MC}_{\text{sat}}$ 为饱和含水量，$\beta$ 为衰减速率参数，$\gamma$ 为曲线形状参数，$\epsilon$ 为最小RCI残余值。该方程满足边界条件（MC = MC$_{\text{dry}}$ 时 RCI = RCI$_{\text{dry}}$），且在MC趋近饱和时不产生负值。

5.9.2 物理信息RCI建模（Physics-Informed RCI）

将Terzaghi承载力理论与RCI经验公式融合。内聚力 $c$ 与含水量MC的关系基于Mohr-Coulomb准则：

$$c = c_0 \cdot \exp\left[-\alpha \cdot \frac{\text{MC} - \text{MC}_0}{\text{MC}_{\text{sat}} - \text{MC}_0}\right]$$

将RCI与Terzaghi极限承载力 $q_u$ 关联：$\text{RCI} = k \cdot q_u$。这种方法将经验公式嵌入物理框架中，使参数具有明确物理含义。

5.9.3 连续化规则体系替代离散规则矩阵

针对Pundir离散规则矩阵导致边界突变的问题，用连续函数替代：

$$f_{\text{slope}}(\theta) = \frac{1}{1 + \exp[k(\theta - \theta_c)]}$$

综合通行性指数采用乘法模型替代加法模型：

$$T = \prod_{i=1}^{n} f_i^{w_i}$$

乘法模型的优势：任一因子为零时总分为零（满足"短板效应"），权重以指数形式影响总分，更符合越野通行性的物理逻辑。

5.9.4 物理-数据融合模型

结合Suvinen的物理建模思想和数据驱动方法：

  1. 使用物理模型计算基础阻力（坡度、承载力、滚动阻力）
  2. 使用机器学习方法（随机森林、XGBoost）学习残差
  3. 总阻力 = 物理预测 + 数据驱动残差

既保持物理可解释性，又利用数据修正模型偏差。

5.9.5 多尺度自适应分析

在不同尺度上使用不同的分析方法：

  • 粗尺度（50-100m）：使用He的规则方法进行全局筛选
  • 中尺度（10-25m）：使用Suvinen的物理模型进行精细分析
  • 细尺度（1-5m）：使用高分辨率数据进行局部优化

需要解决的关键问题：尺度间一致性保证、尺度转换阈值确定、多尺度信息损失控制。

5.9.6 不确定性量化与传播框架

系统地量化和传播模型不确定性：

  • 输入不确定性：使用贝叶斯方法估计参数分布
  • 模型不确定性：使用集成方法（如Bagging）量化模型方差
  • 输出不确定性：使用蒙特卡洛模拟传播不确定性

最终通行性判定应包含置信区间：$P(\text{可通行}) = P(\text{RCI} > \text{VCI})$，当 $P > 0.95$ 时判定为"可通行"，$P < 0.05$ 时判定为"不可通行"。

5.9.7 动态通行性评估框架

建立含水量动态模型与RCI的耦合框架：

$$\text{MC}(t) = \text{MC}_0 + \frac{F(t)}{d_{\text{soil}}}$$

$$\text{RCI}(t) = \exp[a' - b'\ln(\text{MC}(t))]$$

$$\text{Trafficability}(t) = \mathbb{1}[\text{RCI}(t) > \text{VCI}]$$

可预测降雨事件后不同时刻的通行性变化，为行动时间窗口规划提供支持。

5.9.8 深度学习辅助的区域自适应

使用迁移学习实现RCI的区域自适应。网络结构具有物理约束：

  • 输出层使用指数激活函数（确保RCI > 0）
  • 损失函数包含物理约束项：$\mathcal{L}_{\text{monotone}}$（强制RCI随MC单调递减）和 $\mathcal{L}_{\text{boundary}}$（边界条件约束）

在少量目标域数据下即可实现高精度的区域自适应。

5.9.9 万晨CNN方法的改进方向

  • 网络架构升级：使用更深的CNN架构（如ResNet-18）和语义分割替代分块分类
  • 弱监督学习：引入弱监督语义分割（如CAM方法），降低像素级标注成本
  • 边界优化：使用条件随机场（CRF）后处理和多尺度特征金字塔网络（FPN）提高边界精度
  • 知识约束：引入地貌学先验知识（如海拔-起伏度关系）作为网络的归纳偏置

5.10 批判性思考

需要指出的是，本章所识别的"不足"本质上反映了学科发展的阶段性特征——越野通行能力建模处于从经验驱动向数据-物理融合驱动的范式转型期。深度学习等新技术虽展现出强大潜力，但其可解释性不足、对训练数据的强依赖和泛化能力的不确定性，使其在军事安全等高风险场景中的应用面临信任障碍。此外，基础模型虽然有望提供通用的地理空间表征，但越野通行能力建模的特殊性——涉及车辆动力学、土壤力学等专业领域知识——可能限制通用基础模型的直接适用性，领域特异性微调仍不可或缺。

有学者认为，技术发展应遵循"问题驱动"而非"技术驱动"的原则，避免盲目追求算法新颖性而忽视实际应用需求。

6. 方向敏感的越野通行能力建模方法设计

基于前述五章的文献综述与方法论分析，本章提出一种方向敏感的越野通行能力建模方法。该方法的核心创新在于将通行能力从传统的标量场升级为方向敏感的矢量场，从而更真实地反映地形对不同行驶方向车辆的差异化影响。

6.1 问题定义与创新点概述

6.1.1 现有方法的核心问题

通过前述4组文献精读分析，发现现有越野通行能力建模方法存在以下根本性问题：

问题维度具体表现影响
标量化问题通行能力被表示为标量，忽略了行驶方向的影响同一栅格点对不同方向行驶的车辆给出相同代价，不符合实际

6.1.2 创新点设计

核心创新：将通行能力从标量场升级为方向敏感的矢量场

传统方法：$C(x,y) = \text{标量}$（与方向无关）

创新方法：$C(x,y,\theta) = f(\text{地形方向特征}, \text{车辆行驶方向}\,\theta)$

三大创新点：

  1. 方向敏感的坡度阻碍度模型
    • 基于 DEM 梯度分解为纵坡 $g_{\parallel}$（牵引/制动）和横坡 $g_{\perp}$（侧翻/侧滑）
    • 纵坡：上坡受动力限制，陡下坡受制动限制，缓下坡容许
    • 横坡：沿等高线时纵坡≈0但横坡可能很大，不能简单认为"沿等高线最优"
    • 非补偿合成：$R_{\text{slope}} = 1 - (1-R_{\parallel})(1-R_{\perp})$
  2. 地形特征线约束的通行代价修正
    • 利用曲率分析提取山脊线、山谷线
    • 沿特征线方向设置代价折扣
    • 实现"地形引导"的通行能力评估
  3. 基于OSM路网的客观验证方法
    • 利用真实山路数据验证模型合理性
    • 设计多维度验证指标体系

6.2 方向敏感的坡度代价模型

6.2.1 地形梯度与方向分解

梯度向量定义：

对于DEM高程函数 $z = z(x,y)$，梯度向量定义为：

$$\nabla z = \left( \frac{\partial z}{\partial x}, \frac{\partial z}{\partial y} \right) = (p, q) \tag{6-1}$$

其中：

  • $p = \frac{\partial z}{\partial x}$：$x$方向的坡度分量
  • $q = \frac{\partial z}{\partial y}$：$y$方向的坡度分量

梯度的物理意义：

  • 方向：最大坡度方向（最陡上升方向）
  • 大小：$|\nabla z| = \sqrt{p^2 + q^2}$，即坡度值

坡度（slope）：

$$s = \arctan\left(\sqrt{p^2 + q^2}\right) \tag{6-2}$$

坡向（aspect）：梯度方向与$x$轴的夹角

$$\phi = \arctan2(q, p) \tag{6-3}$$

使用$\arctan2$而非$\arctan$的原因：$\arctan2(y,x)$能正确处理四个象限，避免了$\arctan$的象限模糊问题。

车辆行驶方向分解：

设车辆在点$(x,y)$处的行驶方向为$\theta$（相对于$x$轴的角度）。关键洞察：车辆行驶方向与梯度方向的夹角决定了有效坡度。

定义夹角：

$$\alpha = \theta - \phi \tag{6-4}$$

其中：

  • $\alpha = 0°$：车辆沿最大坡度方向上坡
  • $\alpha = 90°$：车辆沿等高线方向行驶
  • $\alpha = 180°$：车辆沿最大坡度方向下坡

6.2.2 有效坡度计算

有效坡度是车辆行驶方向上的坡度分量：

$$s_{\text{eff}}(\theta) = s \cdot \cos(\alpha) = s \cdot \cos(\theta - \phi) \tag{6-5}$$

推导过程：

将梯度向量投影到车辆行驶方向上：

$$\nabla z_{\text{proj}} = |\nabla z| \cdot \cos(\alpha) \tag{6-6}$$

由于$s = \arctan(|\nabla z|)$，对于小坡度有$s \approx |\nabla z|$，因此：

$$s_{\text{eff}} \approx s \cdot \cos(\alpha)$$

物理意义：

  • $\cos(\alpha) > 0$：上坡（有效坡度为正）
  • $\cos(\alpha) = 0$：沿等高线（有效坡度为0）
  • $\cos(\alpha) < 0$：下坡（有效坡度为负）

6.2.3 纵坡与横坡分解（关键修正）

重要修正：仅使用有效坡度 $s_{\text{eff}}$ 存在严重缺陷——它只捕捉了纵向坡度（车辆前进方向），完全忽略了横向坡度（车辆侧向）。当车辆沿等高线行驶时，$s_{\text{eff}} \approx 0$，但横坡可能接近地形实际坡度，导致侧翻、侧滑风险。因此，"沿等高线方向最优"这一表述过于绝对，必须同时考虑横坡。

车辆方向向量定义：

给定行驶方向 $\theta$，定义行驶方向单位向量和横向单位向量：

$$\mathbf{u}_\theta = (\cos\theta, \sin\theta) \tag{6-7a}$$

$$\mathbf{n}_\theta = (-\sin\theta, \cos\theta) \tag{6-7b}$$

纵坡（Longitudinal Slope）：

将梯度投影到车辆行驶方向，得到纵向坡度：

$$g_{\parallel}(x,y,\theta) = \nabla z \cdot \mathbf{u}_\theta = p\cos\theta + q\sin\theta \tag{6-7c}$$

纵坡反映车辆前进方向的牵引/制动需求：$g_{\parallel} > 0$ 为上坡（牵引），$g_{\parallel} < 0$ 为下坡（制动）。

横坡（Cross Slope）：

将梯度投影到车辆横向方向，得到横向坡度：

$$g_{\perp}(x,y,\theta) = \nabla z \cdot \mathbf{n}_\theta = -p\sin\theta + q\cos\theta \tag{6-7d}$$

横坡反映车辆侧向稳定性风险。左右方向对风险基本对称，因此取绝对值 $|g_{\perp}|$。

纵坡/横坡的几何关系：

行驶场景$g_{\parallel}$$g_{\perp}$风险来源
沿最大坡降方向上坡$= |\nabla z|$（最大）$= 0$牵引力不足
沿等高线方向$\approx 0$$\approx |\nabla z|$（最大）侧翻/侧滑
沿最大坡降方向下坡$= -|\nabla z|$（最大）$= 0$制动/失稳
斜向行驶（45°）$= |\nabla z|/\sqrt{2}$$= |\nabla z|/\sqrt{2}$双重风险

合成 DEM 验证：在80×80合成DEM上，对 $\theta=0°$ 方向，在坡度 >10° 的区域实测 $|g_{\parallel}|$ 均值为 0.163，$|g_{\perp}|$ 均值为 0.161，横坡/纵坡比为 0.99，证明横坡与纵坡量级相当，不可忽略。

6.2.4 阻碍度函数设计

基于纵坡和横坡，分别设计阻碍度函数，再通过非补偿合成得到综合方向坡度阻碍度。

（1）纵坡阻碍度 $R_{\parallel} \in [0, 1]$（三段式非对称函数）：

$$ R_{\parallel} = \begin{cases} 1 - \exp\left[-\left(\dfrac{g_{\parallel}}{\alpha_u}\right)^2\right], & g_{\parallel} > 0 \text{（上坡）} \\[8pt] 0, & -\alpha_m \le g_{\parallel} \le 0 \text{（缓下坡）} \\[8pt] 1 - \exp\left[-\left(\dfrac{|g_{\parallel}| - \alpha_m}{\alpha_d}\right)^2\right], & g_{\parallel} < -\alpha_m \text{（陡下坡）} \end{cases} \tag{6-7e} $$

参数含义：

  • $\alpha_u$：上坡阻碍尺度（建议初值15°或研究区坡度P75）
  • $\alpha_m$：缓下坡容许区间（建议3°~5°），在此范围内下坡不增加阻碍
  • $\alpha_d$：陡下坡风险尺度（建议15°~20°）

设计理由：

  • 指数饱和：阻碍度限制在 $[0,1]$，小坡度增长慢，大坡度快速趋近1
  • 缓下坡容许：轻微下坡不增加阻碍（甚至可能降低能耗），陡下坡才增加制动/失稳风险
  • 上坡/下坡不对称：上坡受动力限制，下坡受制动和稳定性限制，物理机制不同

（2）横坡阻碍度 $R_{\perp} \in [0, 1]$：

$$R_{\perp} = 1 - \exp\left[-\left(\frac{|g_{\perp}|}{\alpha_r}\right)^2\right] \tag{6-7f}$$

其中 $\alpha_r$ 为横坡风险尺度（建议8°~12°或横坡P65）。横坡对稳定性更敏感，$\alpha_r$ 应小于或接近 $\alpha_u$。

（3）综合方向坡度阻碍度（非补偿合成）：

$$R_{\text{slope}}(x,y,\theta) = 1 - (1 - R_{\parallel})(1 - R_{\perp}) \tag{6-7g}$$

为什么用非补偿合成而非直接相加？

  • 直接相加 $R_{\parallel} + R_{\perp}$ 可能超过1，需要重新归一化
  • $1 - (1-R_{\parallel})(1-R_{\perp})$ 表示两个风险源的非补偿合成：只要一个风险很高，综合阻碍度就很高
  • 数学性质：$R_{\text{slope}} \in [0, 1]$，且 $R_{\text{slope}} \ge \max(R_{\parallel}, R_{\perp})$

（4）从阻碍度转为边代价：

$$C_{\text{edge}}(x,y,\theta) = l_\theta \cdot \left[-\ln(1 - R_{\text{slope}}(x,y,\theta) + \varepsilon)\right] \tag{6-7h}$$

其中 $l_\theta$ 为方向距离（正交方向为像元大小，斜向为 $\sqrt{2}$ 倍），$\varepsilon = 10^{-6}$ 为数值稳定项。

物理意义： $1 - R$ 可理解为相对通过能力 $P$，$-\ln(P)$ 将通过能力转换为可沿路径累加的代价。低阻碍时代价接近0，高阻碍时代价快速增大。

6.2.5 旧版代价函数（已修正）

原版本6.2.3使用 $C_s = \exp(\beta \cdot |s_{\text{eff}}|)$ 作为代价函数（公式6-7~6-9），该方法存在以下问题：

  • 只考虑纵坡，忽略横坡
  • 代价值域 $[1, +\infty)$ 与阻碍度 $[0, 1]$ 概念混用
  • 乘法综合模型量纲不统一

已替换为上述纵坡/横坡分解+阻碍度框架。

其中$\beta > 0$是敏感系数。

为什么用指数函数？

  1. 非线性响应：坡度对通行能力的影响是非线性的，大坡度的阻碍效果远大于小坡度
  2. 物理合理性：参考Tobler登山函数（Tobler's hiking function）的指数形式
  3. 数学性质：单调递增、正值、可微

敏感系数$\beta$的确定：

方法一：基于临界坡度确定

设临界坡度$s_{\text{crit}}$为车辆无法通行的坡度阈值（如30°），则：

$$\beta = \frac{\ln(C_{\max})}{s_{\text{crit}}} \tag{6-8}$$

其中$C_{\max}$是最大允许代价（如100）。

方法二：基于文献标定

参考Suvinen (2006)的研究，对于普通越野车辆：$s_{\text{crit}} = 25° \sim 35°$，$\beta \approx 0.1 \sim 0.2$（当$s$以度为单位时）。

上坡/下坡的非对称性：

实际上，上坡和下坡对通行的影响不同。上坡主要受动力限制，下坡主要受制动和稳定性限制。改进的代价函数：

$$C_s(x,y,\theta) = \begin{cases} \exp\left(\beta_{\text{up}} \cdot s_{\text{eff}}\right), & \text{if } s_{\text{eff}} > 0 \text{ (上坡)} \\ 1, & \text{if } s_{\text{eff}} = 0 \\ \exp\left(\beta_{\text{down}} \cdot |s_{\text{eff}}|\right), & \text{if } s_{\text{eff}} < 0 \text{ (下坡)} \end{cases} \tag{6-9}$$

通常$\beta_{\text{down}} < \beta_{\text{up}}$（下坡相对容易，但有稳定性风险）。

6.3 地形特征线约束模型

6.3.1 曲率分析与特征线提取

曲率的数学定义：

地形曲率描述地表的弯曲程度。利用Zevenbergen & Thorne (1987)的偏四次曲面拟合方法（详见第2章），可以计算二阶偏导数：

$$r = \frac{\partial^2 z}{\partial x^2} = 2D, \quad t = \frac{\partial^2 z}{\partial y^2} = 2E, \quad s_{xy} = \frac{\partial^2 z}{\partial x \partial y} = F$$

Hessian矩阵：

$$H = \begin{pmatrix} r & s_{xy} \\ s_{xy} & t \end{pmatrix} \tag{6-10}$$

主曲率（Hessian的特征值）：

$$\kappa_{1,2} = \frac{(r+t) \pm \sqrt{(r-t)^2 + 4s_{xy}^2}}{2} \tag{6-11}$$

主方向（特征向量方向）：

$$\theta_{\text{principal}} = \frac{1}{2} \arctan\left(\frac{2s_{xy}}{r-t}\right) \tag{6-12}$$

物理意义：

  • $\kappa_1$（最大曲率）：地形弯曲最剧烈的方向
  • $\kappa_2$（最小曲率）：地形弯曲最平缓的方向
  • 山脊线：沿$\kappa_1$方向，$\kappa_1 < 0$（凸曲率）
  • 山谷线：沿$\kappa_1$方向，$\kappa_1 > 0$（凹曲率）

方向曲率计算：

对于任意方向$\theta$，法曲率（normal curvature）由Euler公式给出：

$$\kappa_n(\theta) = \kappa_1 \cos^2(\theta - \theta_{\text{principal}}) + \kappa_2 \sin^2(\theta - \theta_{\text{principal}}) \tag{6-13}$$

推导：设主方向为$\mathbf{e}_1, \mathbf{e}_2$，任意方向$\mathbf{v} = (\cos\theta, \sin\theta)$在主方向上的投影为$v_1 = \cos(\theta - \theta_{\text{principal}})$，$v_2 = \sin(\theta - \theta_{\text{principal}})$。代入Euler公式$\kappa_n = \kappa_1 v_1^2 + \kappa_2 v_2^2$即得上述结果。

特征线的定义与识别：

特征线类型定义曲率特征检测条件
山脊线（Ridge）局部高程最大值的连线$\kappa_1 < 0$（凸曲率），且$|\kappa_1|$较大$\kappa_1 < -\kappa_{\text{threshold}}$
山谷线（Valley）局部高程最小值的连线$\kappa_1 > 0$（凹曲率），且$|\kappa_1|$较大$\kappa_1 > \kappa_{\text{threshold}}$
等高线（Contour）相同高程的连线方向：垂直于梯度方向—

6.3.2 特征线方向的通行优势

物理意义分析：

  1. 沿山脊线行驶：坡度相对平缓（山脊是分水岭）；视野开阔，便于导航；地面相对坚实（排水良好）
  2. 沿山谷线行驶：坡度相对平缓；可能有水源，但地面可能泥泞；需要结合土壤数据判断
  3. 沿等高线行驶：坡度为0，最省力；但可能遇到横向坡度（侧倾）

6.3.3 特征线约束的代价修正

定义到最近特征线的距离：

$$d_{\text{ridge}}(x,y) = \min_{(x_r,y_r) \in \text{Ridge}} \sqrt{(x-x_r)^2 + (y-y_r)^2} \tag{6-14}$$

$$d_{\text{valley}}(x,y) = \min_{(x_v,y_v) \in \text{Valley}} \sqrt{(x-x_v)^2 + (y-y_v)^2}$$

沿特征线方向的代价折扣：

$$C_{\text{feature}}(x,y,\theta) = 1 - \gamma \cdot \exp\left(-\frac{d_{\text{feature}}^2}{2\sigma^2}\right) \cdot |\cos(\theta - \theta_{\text{feature}})| \tag{6-15}$$

其中：

  • $\gamma \in (0,1)$：最大折扣系数
  • $\sigma$：影响范围参数
  • $\theta_{\text{feature}}$：特征线方向
  • $\cos(\theta - \theta_{\text{feature}})$：行驶方向与特征线方向的一致性

物理意义：越靠近特征线，折扣越大；行驶方向与特征线方向越一致，折扣越大。

分别定义山脊线和山谷线的修正因子：

$$C_{\text{ridge}}(x,y,\theta) = 1 - \gamma_r \cdot \exp\left(-\frac{d_{\text{ridge}}^2}{2\sigma_r^2}\right) \cdot |\cos(\theta - \theta_{\text{ridge}})| \tag{6-16}$$

$$C_{\text{valley}}(x,y,\theta) = 1 - \gamma_v \cdot \exp\left(-\frac{d_{\text{valley}}^2}{2\sigma_v^2}\right) \cdot |\cos(\theta - \theta_{\text{valley}})| \tag{6-17}$$

6.4 综合通行能力模型

6.4.1 单要素代价函数汇总

要素代价函数参数说明
坡度$C_s = \exp(\beta \cdot |s_{\text{eff}}|)$$\beta$: 敏感系数方向敏感
曲率$C_c = 1 + \lambda \cdot |\kappa_n(\theta)|$$\lambda$: 曲率权重方向敏感
地表覆盖$C_l = w_l(x,y)$查表确定与方向无关
土壤承载力$C_{\text{soil}} = f(\text{RCI})$RCI计算与方向无关
水系障碍$C_w = \exp(-\mu \cdot d_w)$$\mu$: 衰减系数与方向无关

6.4.2 乘法综合模型

综合代价函数：

$$C_{\text{total}}(x,y,\theta) = C_s \cdot C_c \cdot C_l \cdot C_{\text{soil}} \cdot C_w \cdot C_{\text{ridge}} \cdot C_{\text{valley}} \tag{6-18}$$

为什么用乘法而非加法？

  1. 物理合理性：任何单一要素的极端值都应导致通行失败，乘法能体现这种"一票否决"特性
  2. 数学性质：乘法模型的结果对各因子的变化更敏感，能更好区分不同区域
  3. 文献支持：参考Pundir (2022)的乘法评分模型

6.4.3 权重确定方法

AHP层次分析法：

步骤1：构建判断矩阵

邀请专家对各要素进行两两比较，构建判断矩阵$A$：

$$A = \begin{pmatrix} 1 & a_{12} & \cdots & a_{1n} \\ 1/a_{12} & 1 & \cdots & a_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ 1/a_{1n} & 1/a_{2n} & \cdots & 1 \end{pmatrix}$$

步骤2：计算权重向量

求解$Aw = \lambda_{\max} w$，归一化得权重向量$w$。

步骤3：一致性检验

$$\text{CI} = \frac{\lambda_{\max} - n}{n - 1}, \quad \text{CR} = \frac{\text{CI}}{\text{RI}} < 0.1 \tag{6-19}$$

熵权法：

步骤1：数据标准化

$$r_{ij} = \frac{x_{ij} - \min_j x_{ij}}{\max_j x_{ij} - \min_j x_{ij}}$$

步骤2：计算信息熵

$$H_j = -\frac{1}{\ln n} \sum_{i=1}^{n} p_{ij} \ln p_{ij}, \quad p_{ij} = \frac{r_{ij}}{\sum_{i=1}^{n} r_{ij}}$$

步骤3：计算权重

$$w_j = \frac{1 - H_j}{\sum_{k=1}^{m} (1 - H_k)} \tag{6-20}$$

6.5 路径规划与代价函数应用

6.5.1 方向敏感的A*算法

传统A*算法的代价函数是标量，需要修改为方向敏感的形式。

代价函数修改：

从节点$(x_1,y_1)$到相邻节点$(x_2,y_2)$的移动代价：

$$g(n) = g(n-1) + C_{\text{total}}(x_1, y_1, \theta_{12}) \cdot d \tag{6-21}$$

其中：

  • $\theta_{12} = \arctan2(y_2-y_1, x_2-x_1)$：移动方向
  • $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$：移动距离

启发式函数设计：

启发式函数需要估计从当前节点到目标的最小代价：

$$h(n) = C_{\min} \cdot d_{\text{Euclidean}}(n, \text{goal}) \tag{6-22}$$

其中$C_{\min}$是所有栅格点的最小代价（通常为1或略大于1）。

可采纳性条件：$h(n)$必须不高估实际代价，即$h(n) \leq h^*(n)$。

6.5.2 代价场可视化

方向采样策略：

在每个栅格点上，采样$N$个方向（如8个或16个），计算每个方向的代价，形成"代价玫瑰图"：

$$\theta_k = \frac{2\pi k}{N}, \quad k = 0, 1, \ldots, N-1$$

可视化方法：

  • 绘制每个栅格点的代价玫瑰图
  • 用颜色表示代价大小
  • 用箭头长度表示方向偏好

6.7 导师创新点的深层理解

6.7.1 从驾驶经验角度理解

老司机的直觉：有经验的越野司机会自然地沿着山脊走，而不是直接翻越山坡。

原因分析：

  • 沿山脊走：坡度相对平缓，不会遇到陡坡
  • 沿等高线走：横向坡度为0，车辆不会侧翻
  • 垂直等高线走：需要爬陡坡，费油且危险

定量验证需求：导师希望用数据证明这个直觉是正确的，需要建立数学模型来解释为什么这些线是"好通行的路"。

6.7.2 物理机制分析

为什么山脊线是好通行的路？

  1. 坡度约束：山脊是分水岭，两侧坡度向山谷倾斜，沿山脊走坡度最小
  2. 曲率约束：山脊线是地形的"脊柱"，沿山脊方向的曲率最小，坡度变化小，车辆行驶更稳定
  3. 地形连续性：山脊线是地形的连续骨架，沿山脊走地形特征连续变化

为什么等高线平行方向是好通行的路？

  1. 坡度为零：等高线方向与梯度垂直，沿等高线走坡度分量为0
  2. 能量最优：上坡需要克服重力做功，沿等高线走不需要做功克服重力

6.7.3 创新点的数学表达

创新点1：方向敏感的坡度分析

传统方法：$C_s = f(s)$（只与坡度大小有关）

创新方法：$C_s = f(s, \theta)$（与坡度大小和方向都有关）

$$C_s(x,y,\theta) = \exp\left(\beta \cdot |s \cdot \cos(\theta - \phi)|\right)$$

创新点2：地形特征线的引导作用

传统方法：不考虑特征线

创新方法：特征线方向有折扣

$$C_{\text{feature}}(x,y,\theta) = 1 - \gamma \cdot e^{-d^2/2\sigma^2} \cdot |\cos(\theta - \theta_{\text{feature}})|$$

创新点3：从标量场到矢量场

传统方法：$C(x,y)$（标量，与方向无关）

创新方法：$C(x,y,\theta)$（矢量场，与方向有关）

核心要义

这就是导师说的"通行能力图就不是静态的或标量的了"的含义——通行能力不再是一个固定的数值，而是随行驶方向变化的方向性函数。

6.8 验证指标设计详解

6.8.1 指标设计的总体思路

核心问题：如何客观评价通行能力评估结果的好坏？

设计原则：

  1. 客观性：基于真实数据（OSM路网），不依赖主观判断
  2. 可量化：有明确的计算公式
  3. 可解释：指标有明确的物理意义
  4. 可对比：不同方法的结果可以用同一指标比较

6.8.2 代价比（Cost Ratio）

$$\text{CR} = \frac{\bar{C}_{\text{OSM}}}{\bar{C}_{\text{random}}} \tag{6-23}$$

物理意义：

  • $\text{CR} < 1$：OSM道路的代价低于随机路径 → 模型合理
  • $\text{CR} \approx 1$：OSM道路没有明显优势 → 模型可能有问题
  • $\text{CR} > 1$：OSM道路的代价高于随机路径 → 模型很可能有问题

为什么设计这个指标？直觉验证：真实道路应该是"好走的路"，代价应该较低；客观性：基于真实数据，不依赖主观判断；可比性：不同方法可以用同一指标比较。

预期结果：好的模型 $\text{CR} \approx 0.5 \sim 0.7$。

6.8.3 排名百分位（Rank Percentile）

$$\text{RP} = \frac{\text{OSM道路代价的排名}}{\text{所有路径总数}} \times 100\% \tag{6-24}$$

物理意义：

  • $\text{RP} < 30\%$：OSM道路属于低代价路径（前30%）→ 模型好
  • $\text{RP} \approx 50\%$：OSM道路处于中等水平 → 模型一般
  • $\text{RP} > 70\%$：OSM道路属于高代价路径 → 模型差

为什么设计这个指标？鲁棒性：不受极端值影响；直观性：百分位更容易理解；统计意义：基于大量随机样本。

6.8.4 方向一致性（Direction Consistency）

$$\text{DC} = \frac{1}{N} \sum_{i=1}^{N} |\cos(\theta_{\text{road},i} - \theta_{\text{optimal},i})| \tag{6-25}$$

物理意义：

  • $\text{DC} \approx 1$：OSM道路方向与最优方向高度一致 → 模型能正确预测最优方向
  • $\text{DC} \approx 0.7$：有一定一致性 → 模型基本合理
  • $\text{DC} \approx 0$：方向随机 → 模型无法预测最优方向

为什么设计这个指标？验证方向敏感性：这是本研究的核心创新点；直接验证：验证"方向敏感模型"是否真的能预测最优方向；区分度高：标量模型的DC应该接近0。

6.8.5 特征线重合率（Feature Line Coincidence）

$$\text{FLC} = \frac{1}{N} \sum_{i=1}^{N} \mathbb{1}(d_{\text{feature},i} < d_{\text{threshold}}) \tag{6-26}$$

物理意义：

  • $\text{FLC} > 0.7$：大部分OSM道路靠近特征线 → 假设成立
  • $\text{FLC} \approx 0.5$：约一半靠近特征线 → 假设部分成立
  • $\text{FLC} < 0.3$：大部分远离特征线 → 假设不成立

为什么设计这个指标？验证核心假设：验证"山脊山谷线是好通行的路"这个假设；解释性：如果FLC高，说明模型的特征线约束是合理的；创新点验证：直接验证创新点2。

9.2.2 合成DEM的数学构造

合成DEM由5个地形分量叠加而成，模拟真实山地环境的基本特征：

（1）基础坡面 — 模拟从西北向东南的整体倾斜：

$$z_{\text{base}}(x, y) = 1000 + 0.02 \cdot x \cdot \Delta + 0.01 \cdot y \cdot \Delta$$

其中 $\Delta = 30\text{m}$ 为栅格大小，$x, y$ 为栅格列号和行号。基础坡面的坡度约为 $\arctan(0.02) \approx 1.15°$，属于微起伏地形。

（2）主山脊 — 模拟东北-西南走向的主山脊线：

$$z_{\text{ridge1}}(x, y) = 80 \cdot \exp\left(-\frac{d_1^2}{2 \sigma_1^2}\right)$$

其中 $d_1 = |0.6(x - 0.3N_c) - 0.4(y - 0.3N_r)|$ 为到山脊线的距离，$\sigma_1 = 0.15 N_c$ 控制山脊宽度，$N_c = 80$ 为列数。主山脊最大高程约80m，位于栅格西南区域。

（3）次山脊 — 模拟另一条较矮的山脊：

$$z_{\text{ridge2}}(x, y) = 50 \cdot \exp\left(-\frac{d_2^2}{2 \sigma_2^2}\right)$$

其中 $d_2 = |0.3(x - 0.7N_c) + 0.7(y - 0.6N_r)|$，$\sigma_2 = 0.1 N_c$。

（4）山谷 — 模拟中间区域的谷地：

$$z_{\text{valley}}(x, y) = -30 \cdot \exp\left(-\frac{d_3^2}{2 \sigma_3^2}\right)$$

其中 $d_3 = |0.5(x - 0.5N_c) + 0.5(y - 0.5N_r)|$，$\sigma_3 = 0.12 N_c$。

（5）随机噪声 — 模拟小尺度地形起伏：

$$z_{\text{noise}}(x, y) \sim \mathcal{N}(0, 5)$$

标准差5m，模拟真实DEM中的微地形变化。

合成DEM总公式：

$$z(x, y) = z_{\text{base}}(x, y) + z_{\text{ridge1}}(x, y) + z_{\text{ridge2}}(x, y) + z_{\text{valley}}(x, y) + z_{\text{noise}}(x, y)$$

结论

本报告围绕"基于地理信息数据的越野通行能力建模分析方法"这一课题，从研究背景与标准化框架、单要素量化建模、多因子综合评估、路径规划应用到现有不足与前沿趋势五个层面展开了系统性综述。综合各章研究发现，可以提炼以下核心判断：

第一，标准化框架已从经验模型迈向物理-数据融合模型。 北约下一代参考机动性模型（NG-NRMM）将预测误差从75.1%大幅降低至6.2%，标志着建模范式的根本性转变 (NG-NRMM)。然而，这一框架对详细车辆参数的高度需求限制了其通用性。

第二，多因子综合评估方法体系已趋于成熟，但权重确定的主观性仍是核心瓶颈。 AHP与熵权法的组合赋权可部分缓解这一矛盾 (MCDA评估方法)，但权重的区域适用性和车辆类型泛化问题仍未得到根本解决。深度学习方法（如LGMM多尺度预测）通过自动学习特征权重提供了新的解决路径 (LGMM)。

第三，路径规划应用已形成从代价图构建到智能优化的完整技术链，但与通行能力建模的集成度不足。 改进A*算法 (动态通行能力图)、改进蚁群算法 (改进蚁群路径规划) 和Pareto多目标优化 (Pareto路径规划) 在越野场景中展现了各自的适用性，但动态重规划和车辆-环境耦合仍有较大提升空间。

第四，全遥感反演方法正在重塑数据获取范式。 武汉大学研究团队提出的全链条遥感反演方法实现了从卫星数据到车辆通行能力的直接转换，车速预测精度优于90% (全遥感反演)，为大规模、无接触式通行能力评估开辟了新途径。

第五，地理空间基础模型是最值得关注的新兴方向。 2021至2024年间已涌现58个遥感视觉基础模型 (地理空间基础模型)，其预训练-微调范式有望从根本上改变当前依赖手工特征工程的研究模式。

展望未来，越野通行能力建模应重点攻关五个方向：动态实时通行能力图、车辆-环境深度耦合模型、不确定性感知的鲁棒路径规划、跨领域知识迁移以及统一基准与标准化评估体系。技术发展应遵循"问题驱动"原则，任何方法创新都应以提升实际决策质量为根本导向。
