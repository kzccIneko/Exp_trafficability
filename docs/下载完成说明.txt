分析区域:
- 经度: 100.846°E ~ 101.124°E
- 纬度: 29.876°N ~ 30.154°N
- 位置: 四川省甘孜州雅江县

============================================================
已成功下载的数据
============================================================

1. Copernicus DEM GLO-30 (30m分辨率)
   文件: downloaded_data/copernicus_dem_30m.tif
   大小: ~39MB
   说明: 高程数据，可用作地形分析基础

2. OpenStreetMap 水系数据
   文件: downloaded_data/osm_water.osm
   大小: ~775KB
   说明: 包含河流、湖泊等水体要素

3. OpenStreetMap 道路数据
   文件: downloaded_data/osm_roads.osm
   大小: ~1.1MB
   说明: 包含所有道路类型（高速、主干道、小路等）

============================================================
需要手动下载的数据
============================================================

4. 12.5m 高分辨率 DEM (ALOS PALSAR)
   下载地址: https://www.eorc.jaxa.jp/ALOS/en/dataset/palsar_l2.htm
   需要瓦片: N029E100
   说明: 需要注册JAXA账号并申请数据访问权限

5. 土地覆盖数据 (ESA WorldCover 2021)
   下载地址: https://esa-worldcover.org/en/data-access
   说明: 10m分辨率，包含11种土地覆盖类型
   需要手动选择覆盖分析区域的瓦片

6. 水系补充数据 (HydroSHEDS)
   下载地址: https://www.hydrosheds.org/
   推荐数据: HydroRIVERS (河流矢量), HydroBASINS (流域边界)

7. 土壤湿度数据 (ERA5-Land)
   下载地址: https://cds.climate.copernicus.eu/
   需要: 注册CDS账号，安装cdsapi
   详见: downloaded_data/cdsapirc_example.txt

============================================================
使用说明
============================================================

OSM数据转换:
- 运行 convert_osm_data.py 可将OSM数据转换为Shapefile格式
- 需要安装: pip install osmium geopandas

详细下载指南:
- 查看 downloaded_data/数据下载指南.md 获取完整说明

============================================================
数据整合建议
============================================================

1. 坐标系统一: 使用 EPSG:4326 (WGS84) 或 EPSG:32647 (UTM 47N)

2. 分辨率统一:
   - 如使用12.5m DEM，需将30m DEM重采样
   - 土地覆盖(10m)可直接使用或重采样到DEM分辨率

3. 数据裁剪:
   - 使用QGIS或ArcGIS裁剪到分析区域边界
   - 边界框: 100.846, 29.876, 101.124, 30.154

4. 格式转换:
   - 栅格: GeoTIFF (.tif)
   - 矢量: Shapefile (.shp) 或 GeoJSON (.geojson)
